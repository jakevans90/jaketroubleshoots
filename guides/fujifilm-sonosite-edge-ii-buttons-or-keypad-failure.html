Fujifilm Sonosite Edge II

Buttons or Keypad Failure

Asset Type

Ultrasound System

Manufacturer

Fujifilm Sonosite

Model

Edge II

What This Guide Helps With

This guide assists Clinical Engineering in troubleshooting situations where the Fujifilm Sonosite Edge II’s physical buttons or keypad fail to respond. Common symptoms include:

  • No response when pressing keys or buttons
  • Intermittent functionality of certain keys
  • System only responding to touchscreen inputs (if available)

The guide emphasizes checking external and user-interaction issues before assuming internal keypad failure.

Step-by-Step Troubleshooting

Confirm System Power

  • Ensure the ultrasound system is powered on and the battery has sufficient charge or the unit is connected to AC power.
  • Why: A low battery or unstable power can cause input lag or unresponsive keys.

Visual Inspection of Keypad

  • Check for visible debris, liquid contamination, or damage around keys.
  • Why: Dirt, dust, or spills can prevent proper key contact.

Clean the Keypad

  • Use a lint-free cloth lightly dampened with isopropyl alcohol to clean around and on the keys. Avoid excessive liquid.
  • Why: Cleaning removes dirt and oils that may block electrical contact.

Check for External Key Lock or Function Mode

  • Verify that any “lock” or “function mode” settings on the system are not engaged (refer to system touchscreen menus).
  • Why: Some ultrasound systems allow locking buttons to prevent accidental presses.

Test All Keys Methodically

  • Press each button or key individually while observing system response. Note any keys that are non-responsive or inconsistent.
  • Why: Pinpoints which buttons are failing and helps rule out software vs. hardware issues.

Restart the System

  • Perform a controlled system reboot.
  • Why: Sometimes software glitches can temporarily disable keypad input.

Check for Firmware Updates

  • Ensure the device firmware is current per manufacturer guidance.
  • Why: Firmware updates can resolve known input or keypad issues.

If the Problem Persists

External checks (power, cleaning, mode settings) have been ruled out.

Keypad or buttons may have an internal failure.

  • Remove the system from clinical service.
  • Label the unit as Out of Service.
  • Send the system to a qualified repair vendor or in-house bench evaluation.

Knowing when to stop and escalate prevents potential further damage and ensures patient safety.

Clinical Use Tip

  • Do not attempt patient imaging if buttons are unreliable.
  • Move the patient to another functioning ultrasound system before troubleshooting further.
  • Only perform keypad troubleshooting when the unit is safely powered off or not in active clinical use.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
“Several keys on the Edge II keypad do not respond, and imaging cannot be fully controlled.”

Cause

What was observed during troubleshooting.

Example:
External inspection revealed no liquid damage. Keypad cleaning and system restart did not restore functionality. Likely internal keypad failure.

Resolution

What action was taken.

Example:
Device removed from service, labeled Out of Service, and sent for vendor repair.

Helpful Details to Include

  • Keys tested individually
  • Any keys intermittently responsive
  • System power status during testing
  • Firmware version
  • Any visible debris or damage

Final Thought

Patient safety and reliable system operation are paramount. Stepwise troubleshooting helps determine when an issue is externally fixable versus requiring repair. Accurate documentation ensures continuity and compliance.

That is successful troubleshooting.
