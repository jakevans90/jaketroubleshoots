Drager Evita V500

Alarm: Flow sensor issue airflow measurement inaccurate or unavailable

Asset Type

Ventilator

Manufacturer

Drager

Model

Evita V500

What This Guide Helps With

This guide addresses the alarm condition where the ventilator reports a flow sensor issue, indicating inaccurate or unavailable airflow measurement. This can result in unreliable tidal volume delivery and compromised ventilation monitoring.

Step-by-Step Troubleshooting

1. Ensure Patient Safety First

  • Immediately assess the patient’s ventilation status
  • If ventilation accuracy is in question, remove from ventilator
  • Provide alternate ventilation (bag-valve mask or backup ventilator)

2. Verify the Reported Alarm and Behavior

  • Confirm “Flow sensor issue” is displayed
  • Observe when the alarm occurs (startup, during use, intermittent)
  • Check for inconsistent or missing flow/volume readings
  • Look for erratic tidal or minute volume values

3. Check Flow Sensor for Contamination or Blockage

  • Remove and inspect the flow sensor
  • Look for moisture, secretions, or debris
  • Clean per facility protocol if contamination is present

4. Verify Proper Sensor Installation

  • Ensure correct orientation of the sensor
  • Confirm secure connection
  • Reseat the sensor if needed

5. Check Breathing Circuit Condition

  • Inspect for condensation near the sensor
  • Check tubing for kinks or obstructions
  • Ensure unobstructed airflow through the circuit

6. Perform Flow Sensor Calibration / System Test

  • Run System Self Test (SST)
  • Follow prompts for calibration
  • Verify test completes successfully

7. Swap with Known Good Sensor (If Available)

  • Replace with a known good flow sensor
  • Re-run system test
  • Confirm alarm clears

8. Stop When Resolved

  • Return device to service if alarm clears
  • Verify stable and accurate ventilation readings

If the Problem Persists

If the alarm continues after cleaning, calibration, and sensor replacement, external causes have been ruled out and an internal failure is likely.

  • Remove the device from service
  • Label as Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot on an active patient. Move the patient to a backup device first before addressing flow sensor issues.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Ventilator alarming 'Flow sensor issue' with inconsistent tidal volume readings."

Cause

What was observed during troubleshooting.

Example:
"Flow sensor contaminated with moisture causing inaccurate airflow measurement."

Resolution

What action was taken.

Example:
"Flow sensor cleaned and recalibrated. System test passed and alarm cleared."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Flow sensor accuracy is critical for safe ventilation delivery. Always begin with external causes like contamination and setup, and escalate appropriately if the issue persists.

That is successful troubleshooting.
