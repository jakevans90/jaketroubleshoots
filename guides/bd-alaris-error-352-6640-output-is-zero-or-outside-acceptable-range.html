BD Alaris

Error: 352.6640 Output is zero or outside acceptable range

Asset Type

Infusion Pump

Manufacturer

BD

Model

Alaris

What This Guide Helps With

This guide addresses Error 352.6640 indicating the force sensor output is zero or outside acceptable range. This condition prevents accurate pressure monitoring, which can lead to undetected occlusions or unsafe infusion delivery.

Step-by-Step Troubleshooting

1. Ensure Patient Safety First

  • Remove the pump from active patient use
  • Transfer infusion to a known working device
  • Verify therapy continues without interruption

2. Verify the Reported Error and Behavior

  • Power on the pump and confirm Error 352.6640 is displayed
  • Observe when the error occurs (startup, loading, or infusion)
  • Check for abnormal or absent pressure readings

3. Check IV Set Installation

  • Remove and reinstall the IV set
  • Ensure correct BD-approved Alaris set is used
  • Verify proper routing and full seating in the channel
  • Confirm door fully closes and latches

4. Inspect Pumping Channel and Sensor Area

  • Inspect for fluid contamination, residue, or debris
  • Check for dried medication buildup
  • Ensure no obstruction in the sensor contact area
  • Clean per approved procedures if needed

5. Try a Known-Good IV Set

  • Load a new, known-good administration set
  • Attempt to operate the pump
  • Observe if error clears

6. Power Cycle the Pump

  • Turn off the pump
  • Wait 10–15 seconds
  • Power back on and observe for error

7. Check for Door or Latch Issues

  • Inspect door alignment and closure
  • Check latch for wear or looseness
  • Look for warping or mechanical misalignment

8. Perform Basic Functional Test

  • If error clears, run a test infusion off patient
  • Monitor for recurrence
  • Confirm stable operation

If the Problem Persists

If the error continues after verifying setup, cleaning, and replacing the IV set, external causes have been ruled out. This indicates a likely internal force sensor or circuitry failure.

  • Remove device from service
  • Label as Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot infusion pumps on an active patient. Always transfer the patient to a backup device before addressing pressure or sensor-related issues.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"BD Alaris pump displaying Error 352.6640; infusion would not start."

Cause

What was observed during troubleshooting.

Example:
"Force sensor output remained zero/out of range after verifying proper set installation and replacing IV set."

Resolution

What action was taken.

Example:
"Removed device from service and sent for repair due to suspected internal sensor failure."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Force sensor errors directly impact occlusion detection and infusion safety. Always rule out setup and disposable issues first, but escalate promptly when the error persists.

That is successful troubleshooting.
