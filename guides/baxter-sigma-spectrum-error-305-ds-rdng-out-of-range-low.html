Baxter SIGMA Spectrum

Error: 305 DS rdng out of range (low)

Asset Type

Infusion Pump

Manufacturer

Baxter

Model

SIGMA Spectrum

What This Guide Helps With

Error 305 DS rdng out of range (low) indicates a low downstream sensor reading that may prevent infusion from starting or continuing. This can be caused by cassette misalignment, improper setup, flow path issues, or a downstream sensor fault.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Remove the pump from active patient use if infusion is interrupted or unreliable
  • Transfer infusion to a functioning pump if therapy is ongoing

Verify the Reported Error and Behavior

  • Power on the pump and confirm Error 305 DS rdng out of range (low) is displayed
  • Observe when the error occurs (startup, set loading, or infusion)
  • Check for symptoms such as inability to start infusion or repeated alarms

Check IV Set Installation

  • Open the pump door and inspect cassette placement
  • Ensure cassette is fully seated and aligned in the channel
  • Verify the door is fully latched
  • Remove and reload the IV set carefully

Inspect IV Set and Cassette Condition

  • Check for cracks, deformities, or damage to the cassette
  • Ensure there is no air in the line near the cassette
  • Verify the set is properly primed
  • Replace with a new, known-good IV set if needed

Check for Flow Obstruction or Open Line

  • Inspect downstream tubing for kinks or clamps
  • Ensure tubing is connected and not open to air
  • Confirm proper priming and fluid path continuity

Clean Sensor Area (External Only)

  • Inspect the pump channel where the cassette sits
  • Look for fluid residue, debris, or contamination
  • Clean using approved methods if necessary

Power Cycle the Pump

  • Turn the pump off
  • Wait a few seconds
  • Restart the pump and check for the error

Isolate Accessory vs Device Issue

  • If the error resolves with a new IV set, the issue is accessory-related
  • If the error persists across multiple sets, the issue is device-related

If the Problem Persists

If external causes such as IV set installation, tubing condition, and environment have been ruled out and Error 305 continues, this indicates a likely internal downstream sensor fault or calibration issue.

  • Remove the device from service
  • Label the device Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot infusion pump cassette or sensor issues while the device is connected to a patient. Always transfer therapy to a backup pump before performing checks.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Pump displaying Error 305 DS rdng out of range (low); infusion would not start."

Cause

What was observed during troubleshooting.

Example:
"Error persisted after multiple IV set replacements and proper loading; suspected downstream sensor fault."

Resolution

What action was taken.

Example:
"Removed from service, labeled out of service, and sent for repair."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Error 305 DS rdng out of range (low) is often caused by setup or cassette issues but can indicate a deeper sensor failure if persistent. Always start with external checks, prioritize patient safety, and escalate appropriately when needed.

That is successful troubleshooting.
