Medtronic Valleylab FT10

Error: E350 Internal Check

Asset Type

Electrosurgical Unit (ESU)

Manufacturer

Medtronic

Model

Valleylab FT10

What This Guide Helps With

This guide helps troubleshoot Error E350 Internal Check on the Medtronic Valleylab FT10 electrosurgical unit. Error E350 indicates a return electrode overtemperature condition, which may interrupt ESU activation until the system cools and safely reactivates.

This condition should be treated as a patient safety-related thermal warning. Possible contributing factors include poor return electrode contact, return electrode cable issues, accessory problems, extended activation, restricted ventilation, or an internal sensing/protection issue.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Do not continue using the ESU on a patient if the unit reports a return electrode overtemperature condition or activation is interrupted.
  • Remove the Valleylab FT10 from active patient use if the error appears during a procedure and the issue cannot be immediately cleared safely.
  • Use another ESU if surgical energy is urgently needed.
  • Do not repeatedly activate the unit while Error E350 is present.

Verify the Reported Error and Behavior

  • Confirm the display shows Error: E350 Internal Check.
  • Confirm whether the reported condition includes Return Electrode Overtemp.
  • Ask staff when the error appeared, such as during activation, after repeated activations, after extended use, immediately at startup, or intermittently with certain accessories or modes.
  • Document whether the error was reproduced or confirmed from staff report.

Allow Cooling Time

  • Stop ESU activation.
  • Allow the system and patient return electrode pathway time to cool.
  • Do not disconnect or move accessories while the surgical field is active unless clinically appropriate.
  • After cooling, attempt activation only when it is safe to do so.
  • If the error clears and normal operation returns, verify function and document the event before returning the device to service.

Check the Return Electrode Pad and Patient Contact

  • Confirm the return electrode pad is properly applied to the patient.
  • Check that the pad has full skin contact.
  • Look for lifting edges, wrinkles, gel drying, poor adhesion, fluid contamination, hair, scar tissue, bony areas, or placement over poor tissue contact.
  • Replace the return electrode pad if there is any doubt about adhesion or condition.
  • Verify that return electrode contact is clean, secure, flat, and appropriate for safe ESU use.

Check the Return Electrode Cable and Connection

  • Confirm the return electrode cable is fully seated in the Valleylab FT10.
  • Inspect the cable for cuts, crushed sections, bent pins, loose connectors, contamination, or fluid intrusion.
  • Reseat the return electrode cable connection.
  • If available, test with a known-good return electrode cable or pad setup.

Check for Accessory or Instrument-Related Conditions

  • Ask staff which instrument was being used when the error occurred.
  • Inspect the active accessory for damage, contamination, sticking activation buttons, or unusual heating.
  • Try a known-good compatible instrument if safe and appropriate.
  • Do not keep using an accessory that appears damaged or overheated.
  • Determine whether the problem follows one accessory setup or remains with the ESU.

Review Use Pattern and Output Settings

  • Ask whether the error occurred after long or repeated activations.
  • Review the active mode, power setting, and clinical workflow.
  • Confirm the settings are appropriate for the instrument and procedure.
  • Allow additional cooling time if the error followed extended activation.

Inspect the ESU Externally

  • Check the Valleylab FT10 for blocked vents, dust buildup, fluid contamination, physical damage, unusual odor, abnormal heat, or fan concerns.
  • Verify the unit is not covered by drapes or positioned where ventilation is restricted.
  • Confirm the ESU is connected to proper AC power.
  • Remove the unit from service if there are signs of overheating, contamination, damage, or abnormal operation.

Power Cycle and Recheck

  • Only power cycle the ESU if it is safe to interrupt use and the device is no longer needed for active patient care.
  • Turn the unit off.
  • Wait briefly.
  • Turn the unit back on.
  • Allow startup checks to complete.
  • Reconnect a known-good return electrode and accessory setup.
  • Confirm whether Error E350 returns during safe functional checks.

If the Problem Persists

If Error E350 returns after cooling time, proper return electrode placement, cable checks, accessory replacement, settings review, and external inspection, the likely simple external causes have been ruled out.

Remove the Valleylab FT10 from service. Label the unit Out of Service. Do not return it to the OR for patient use. Send the unit for repair or bench evaluation.

Persistent Error E350 may indicate the ESU is detecting a recurring return electrode overtemperature condition or an internal sensing/protection issue that should not be bypassed.

Clinical Use Tip

Never troubleshoot return electrode overtemperature errors on an active patient. Move the patient to a backup device or transfer surgical energy use to a known-good ESU first. Do not ignore a return electrode heating-related error or repeatedly reactivate the unit just to continue the case.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"User reported Medtronic Valleylab FT10 displayed Error E350 Internal Check during activation. Staff reported return electrode overtemperature message and interrupted ESU activation."

Cause

What was observed during troubleshooting.

Example:
"Return electrode overtemperature condition confirmed. Return electrode pad placement, cable connection, accessory condition, ventilation, and use pattern were checked. Error returned after cooling and known-good accessory setup."

Resolution

What action was taken.

Example:
"Allowed cooling time, inspected return electrode pad and cable, checked accessory setup, verified external condition and ventilation, and reactivated safely. Error returned, so unit was removed from service, labeled Out of Service, and sent for bench evaluation."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Error E350 should be handled as a safety-related thermal warning, not just a nuisance message. Start with cooling time, return electrode contact, cables, accessories, settings, and ventilation. If the error persists after external causes are ruled out, remove the ESU from service and escalate appropriately. Clear documentation helps show what was checked, what was found, and why the final decision was safe.

That is successful troubleshooting.
