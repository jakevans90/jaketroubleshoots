BD Alaris

Error: 222.4060 Over-infusion state detected (infusion past VTBI limit)

Asset Type

Infusion Pump

Manufacturer

BD

Model

Alaris

What This Guide Helps With

This guide addresses Error 222.4060 indicating that the infusion exceeded the programmed VTBI limit. This condition causes the pump to stop due to a potential over-infusion event, which can lead to unsafe medication or fluid delivery.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Stop the infusion immediately
  • Remove the pump from patient use
  • Assess patient status and verify infused volume
  • Transfer therapy to a functioning pump if needed

Verify the Reported Error and Behavior

  • Confirm Error 222.4060 is displayed
  • Determine if the pump alarmed and stopped automatically
  • Check if infusion continued past VTBI
  • Note if issue occurred at end-of-infusion or prematurely

Check VTBI Programming and Infusion Setup

  • Review VTBI, infusion rate, and mode
  • Compare programmed VTBI to physician order
  • Verify actual infused volume matches expectations

Inspect Administration Set Loading

  • Ensure tubing is fully seated in the pumping channel
  • Confirm door is fully latched
  • Verify correct BD-compatible set is used

Check for Free-Flow Conditions

  • Verify roller clamp is properly managed
  • Confirm anti-free-flow mechanism is functioning
  • Check for fluid movement when pump is stopped

Check for Upstream and Downstream Conditions

  • Inspect for kinks or occlusions
  • Check for air-in-line conditions
  • Verify proper bag height and setup

Replace Administration Set

  • Install a new, known-good BD administration set
  • Reprogram infusion
  • Observe if error reoccurs

Perform Controlled Functional Test

  • Run a test infusion with known VTBI and rate
  • Verify accurate delivery
  • Confirm pump stops at VTBI

If the Problem Persists

If external causes such as programming errors, tubing issues, and disposable faults have been ruled out, the issue likely indicates an internal control or safety system failure.

  • Remove the device from service
  • Label the device Out of Service
  • Send for repair or manufacturer evaluation

Clinical Use Tip

Never troubleshoot on an active patient. Always move the patient to a backup device before performing any troubleshooting steps.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Pump displayed Error 222.4060 and appeared to infuse past the programmed VTBI."

Cause

What was observed during troubleshooting.

Example:
"Incorrect VTBI programming or possible free-flow condition; internal control fault suspected after verification."

Resolution

What action was taken.

Example:
"Verified settings, replaced administration set, tested unit; removed from service when error persisted."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Over-infusion conditions are high-risk events. Always verify setup and rule out external causes first, but escalate immediately if the pump cannot reliably stop at the programmed VTBI.

That is successful troubleshooting.
