Hillrom Centrella Smart+

Error: 0x11A2 Left Pendant Button Stuck (Moderate)

Asset Type

Hospital Bed

Manufacturer

Hillrom

Model

Centrella Smart+

What This Guide Helps With

Error 0x11A2 indicating a stuck left pendant button causing unintended bed movement or loss of control from the patient pendant.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Remove the bed from active patient use if unintended movement is occurring
  • Transfer the patient to a safe bed if necessary
  • Do not rely on malfunctioning pendant controls

Verify the Reported Error and Behavior

  • Confirm the bed displays Error 0x11A2
  • Observe if the bed moves without input
  • Check if the left pendant is unresponsive or continuously activating a function
  • Note if the error appears at startup or during use

Inspect the Left Pendant for Physical Obstruction

  • Check for debris or dried fluids around buttons
  • Look for physical damage or deformation
  • Ensure buttons return to neutral position when pressed
  • Clean using approved disinfectant

Check Pendant Cable and Strain Relief

  • Inspect cable for kinks, cuts, or pinched areas
  • Verify secure connection at the bed interface
  • Ensure cable is not under tension or trapped

Power Cycle the Bed

  • Turn off or unplug the bed
  • Wait 30–60 seconds
  • Restore power and restart

Isolate Pendant vs Bed Control Issue

  • Connect a known-good pendant if available
  • Test all bed functions
  • Determine if error clears with replacement pendant

Stop When Resolved

  • Clear the error if resolved
  • Verify all pendant functions operate correctly
  • Return the bed to service

If the Problem Persists

If external causes such as contamination, cable damage, and pendant failure have been ruled out, the issue likely involves an internal control or input circuit failure.

  • Remove the bed from service
  • Label as Out of Service
  • Send for repair or further bench evaluation

Clinical Use Tip

Never troubleshoot on an active patient. Move the patient to a backup device before continuing diagnosis or repair.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Bed displaying Error 0x11A2 with left pendant causing continuous or unresponsive input."

Cause

What was observed during troubleshooting.

Example:
"Left pendant button found stuck due to contamination and did not return to neutral position."

Resolution

What action was taken.

Example:
"Cleaned and tested pendant; replaced pendant when issue persisted; verified proper function and returned bed to service."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Error 0x11A2 is typically caused by external factors like contamination or cable damage. Always start with simple checks before escalating to component replacement or repair. Prioritizing patient safety and structured troubleshooting ensures efficient and reliable resolution.
