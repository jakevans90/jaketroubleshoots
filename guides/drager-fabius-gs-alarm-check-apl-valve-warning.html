Drager Fabius GS

Alarm: CHECK APL VALVE !!!

Asset Type

Anesthesia Machine

Manufacturer

Drager

Model

Fabius GS

What This Guide Helps With

Alarm CHECK APL VALVE !!! indicating unintended expiratory flow during inspiration, which can cause improper ventilation, system leaks, or APL/valve control malfunction affecting patient safety.

Step-by-Step Troubleshooting

1. Ensure Patient Safety First

  • Remove the machine from active patient use if ventilation is questionable
  • Transfer the patient to a backup anesthesia machine or manual ventilation
  • Confirm stable ventilation before proceeding

2. Verify the Reported Alarm and Behavior

  • Confirm CHECK APL VALVE !!! alarm is present
  • Observe when it occurs (startup or during ventilation)
  • Check ventilation mode (Volume, Pressure, Pressure Support)
  • Look for abnormal pressure or flow patterns

3. Check Breathing Circuit for External Causes

  • Ensure all hoses are securely connected
  • Check for loose fittings or disconnections
  • Inspect tubing for cracks, holes, or wear

4. Inspect the APL Valve Setting and Function

  • Verify APL valve is set appropriately for current mode
  • Cycle the valve to ensure smooth movement
  • Check for contamination or sticking

5. Check Expiratory Valve Assembly

  • Inspect valve and valve disk for moisture or debris
  • Ensure proper seating of valve components
  • Replace disposable parts if applicable

6. Inspect Expiratory Control Line / Hose

  • Ensure hose is securely connected
  • Check for kinks, damage, or disconnections

7. Check Flow Sensor Condition

  • Inspect for condensation or contamination
  • Ensure proper connection
  • Clean or replace per facility guidelines if needed

8. Perform Flow Sensor Calibration

  • Run calibration procedure from user-accessible menu
  • Confirm calibration completes without error

9. Re-test the System

  • Connect to a test lung
  • Run through ventilation modes
  • Confirm alarm is cleared and ventilation is normal

If the Problem Persists

If external causes have been ruled out, the issue may involve internal valve control, pneumatics, or flow measurement systems.

  • Remove device from service
  • Label Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot breathing system or valve-related alarms while connected to a patient. Always switch to a backup ventilation method first.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Unit alarming CHECK APL VALVE !!! during ventilation with abnormal flow behavior."

Cause

What was observed during troubleshooting.

Example:
"Contamination found in expiratory valve preventing proper sealing."

Resolution

What action was taken.

Example:
"Cleaned/reseated expiratory valve, verified circuit integrity, performed flow sensor calibration, alarm cleared."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

This alarm indicates a critical ventilation integrity issue. Always start with simple external causes, but escalate when system control components are involved to ensure patient safety.

That is successful troubleshooting.
