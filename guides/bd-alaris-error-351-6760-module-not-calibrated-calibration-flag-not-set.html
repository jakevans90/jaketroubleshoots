BD Alaris

Error: 351.6760 Module not calibrated (calibration flag not set)

Asset Type

Infusion Pump

Manufacturer

BD

Model

Alaris

What This Guide Helps With

This guide addresses Error 351.6760 indicating an Alaris module is not calibrated, preventing operation and causing module rejection or system startup failure. This condition means the module cannot ensure accurate infusion delivery and must not be used clinically.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Stop using the affected module immediately
  • Transfer infusion to a functioning pump or module
  • Verify patient therapy continues without interruption

Verify the Reported Error and Behavior

  • Power on the Alaris system
  • Identify which module is attached (pump, syringe, PCA, etc.)
  • Observe when the error appears (startup vs connection)
  • Confirm Error: 351.6760 is displayed

Isolate the Affected Module

  • Remove the module associated with the error
  • Power system with no modules attached
  • Attach a known good module if available
  • Confirm whether the error follows the specific module

Reseat the Module

  • Power down the system
  • Disconnect and reconnect the module securely
  • Ensure proper alignment and full engagement

Inspect Module and Connection Points

  • Check for bent or damaged connector pins
  • Inspect for debris or contamination
  • Look for physical damage to module housing

Attempt System Startup Again

  • Reconnect the module and power on the system
  • Determine if error clears or persists
  • Confirm if issue is connection-related or internal

If the Problem Persists

If Error 351.6760 continues after external checks:

  • External causes have been ruled out
  • Error consistently follows a specific module

Action:

  • Remove the module from service
  • Label the device as Out of Service
  • Send for calibration or repair using approved service procedures

Clinical Use Tip

Never troubleshoot or attempt to use a module with calibration errors on an active patient. Always move the patient to a properly functioning device before addressing the issue.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Alaris module displaying Error 351.6760 and would not allow infusion to start."

Cause

What was observed during troubleshooting.

Example:
"Calibration flag missing or corrupted within module. Issue isolated to specific module; connections verified intact."

Resolution

What action was taken.

Example:
"Removed module from service, labeled Out of Service, and sent for calibration/repair."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Calibration errors on modular infusion systems indicate a loss of critical configuration data. Once external causes are ruled out, immediate removal from service and proper escalation ensures patient safety and maintains system reliability.

That is successful troubleshooting.
