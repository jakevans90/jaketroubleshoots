STERIS AMSCO 3085 SP

Hand Control or Pendant Not Responding

Asset Type

Surgical Table

Manufacturer

STERIS

Model

AMSCO 3085 SP

What This Guide Helps With

This guide assists Clinical Engineering in troubleshooting situations where the STERIS AMSCO 3085 SP surgical table powers on but the hand control or pendant does not operate the table. Common symptoms include:

  • No response when buttons are pressed
  • Indicator lights on the pendant not illuminating
  • Table movement functions inoperative via the hand control

Focus is on external, easily verifiable checks before assuming internal failure.

Step-by-Step Troubleshooting

Ensure Table is Powered On

  • Verify the main power switch is in the ON position.
  • Confirm that table indicator lights are illuminated.

Why: A hand control cannot operate a table that has no power.

Check Pendant Connection

  • Inspect the cord between the hand control/pendant and the table for visible damage, kinks, or loose connections.
  • Disconnect and reconnect firmly.

Why: Loose or damaged connections are the most common cause of unresponsive pendants.

Swap Out Hand Control (If Available)

  • If another functional pendant exists, swap it to see if the issue follows the pendant or remains with the table.

Why: Helps isolate whether the problem is the pendant or the table electronics.

Inspect Fuses and Circuit Breakers

  • Check any external fuses or breakers specific to the table or hand control circuit.
  • Replace blown fuses with manufacturer-approved equivalents.

Why: A blown fuse can prevent the pendant from sending commands.

Verify Emergency Stop / Lockouts

  • Ensure the table’s emergency stop buttons are not engaged.
  • Confirm no service or lockout switches are activated.

Why: Table safety interlocks disable pendant function.

Check for Indicator Lights or Error Codes

  • Observe if the table or pendant displays any error lights when the pendant is pressed.
  • Document codes for reference or escalation.

Why: Codes can indicate specific circuits or control board issues.

If the Problem Persists

All external checks completed without resolution.

Likely internal electronics issue in the table control system or pendant circuitry.

Next Steps:

  • Remove table from service.
  • Label table Out of Service.
  • Send for repair or bench evaluation per STERIS service instructions.

Clinical Engineering Note: Stopping at this point avoids further risk and preserves patient safety.

Clinical Use Tip

Never attempt to troubleshoot pendant or table electronics while a patient is on the table.

Move the patient to an alternative table before further troubleshooting.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
“Table pendant unresponsive; buttons do not control table.”

Cause

What was observed during troubleshooting.

Example:
“Pendant cord intact, table powered on, fuses intact; emergency stop not engaged; pendant remains unresponsive.”

Resolution

What action was taken.

Example:
“Isolated issue to table electronics. Table removed from service and sent for vendor evaluation.”

Helpful Details to Include

  • Outlet tested: Yes / No
  • Pendant cable swapped: Yes / No
  • Indicator lights on pendant: Observed / Not observed
  • Any error codes displayed: List if present
  • Final device status: Out of service, sent for repair

Final Thought

Patient safety and proper documentation are paramount. Following a logical, external-first troubleshooting path ensures minimal risk, identifies avoidable causes, and clearly communicates findings for repair or escalation.

That is successful troubleshooting.
