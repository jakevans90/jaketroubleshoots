Baxter SIGMA Spectrum

Error: 300 Fail to enable US sensor

Asset Type

Infusion Pump

Manufacturer

Baxter

Model

SIGMA Spectrum

What This Guide Helps With

This guide addresses Error 300 Fail to enable US sensor, which indicates failure of the ultrasonic sensor system. This condition prevents the pump from detecting fluid or air-in-line, leading to inability to safely start or continue an infusion.

Step-by-Step Troubleshooting

1. Ensure Patient Safety First

  • Remove the pump from active patient use immediately
  • Transfer infusion to a known working device

2. Verify the Reported Error and Behavior

  • Power on the pump and confirm the error is displayed
  • Note when the error occurs (startup, priming, infusion)
  • Observe for inability to start infusion or immediate alarms

3. Check IV Set Installation and Compatibility

  • Verify correct Baxter-approved IV set is installed
  • Ensure tubing is properly loaded and aligned in the pump channel
  • Confirm proper positioning within the sensor area

4. Inspect the Pump Channel and Sensor Area

  • Open the pump door and inspect for residue or contamination
  • Check for moisture, condensation, or debris
  • Clean using approved wipes if necessary

5. Reseat the IV Set and Close Door Securely

  • Remove and reinstall the IV set carefully
  • Ensure proper alignment and seating
  • Close the pump door firmly until fully latched

6. Power Cycle the Pump

  • Turn the pump off completely
  • Wait 10–15 seconds
  • Power the pump back on

7. Test with a Known-Good IV Set

  • Replace with a new, verified IV set
  • Attempt to start a basic infusion
  • Observe for normal operation without error

If the Problem Persists

If Error 300 continues after all external checks, external causes have been ruled out. This likely indicates internal ultrasonic sensor failure or control circuitry fault.

  • Remove the device from service
  • Label the device as Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot on an active patient. Always move the patient to a backup infusion pump before attempting any troubleshooting steps.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Pump displaying Error 300 ‘Fail to enable US sensor’; unable to initiate infusion."

Cause

What was observed during troubleshooting.

Example:
"Ultrasonic sensor failed to initialize. External checks completed including IV set replacement, cleaning, and power cycle. Issue persisted."

Resolution

What action was taken.

Example:
"Removed pump from service and sent for repair. Replacement pump provided for clinical use."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Ultrasonic sensor failures directly impact patient safety by disabling critical air-in-line detection. Always prioritize external setup and cleanliness checks first, but escalate quickly when sensor faults persist.

That is successful troubleshooting.
