STERIS STERIS AMSCO 4085

Error: POWER DEF

Asset Type

Surgical Table

Manufacturer

STERIS

Model

STERIS AMSCO 4085

What This Guide Helps With

Error: POWER DEF indicating transformer overheating or fuse F2 failure, causing the table to run on battery instead of mains power and potentially leading to loss of functionality during procedures.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Transfer the patient to another table before troubleshooting
  • Remove the device from active use

Verify the Reported Error and Behavior

  • Confirm that “POWER DEF” is displayed
  • Check if the table is operating only on battery power
  • Look for absence of AC power indication

Check Facility Power Connection

  • Ensure the table is plugged into a working outlet
  • Test the outlet with another device

Inspect Power Cord and Plug

  • Check for visible damage or loose connections
  • Reseat the power cord securely

Allow the Table to Cool Down

  • Power off the table and allow time to cool
  • Restart after cooling period

Check Fuse F2 (If Accessible Externally)

  • Inspect fuse F2 for signs of failure
  • Replace with correct type and rating if blown

Confirm Return to Mains Operation

  • Reconnect power and turn the table on
  • Verify AC operation is restored
  • Confirm the error is no longer present

Stop if Resolved

  • Return device to service if functioning normally
  • Monitor for recurrence

If the Problem Persists

If external power, connections, cooling, and fuse F2 have been verified:

  • Internal power supply or transformer failure is likely
  • Remove the device from service
  • Label as Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot on an active patient. Move the patient to a backup device before addressing power-related issues.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Table displaying POWER DEF error and running on battery only."

Cause

What was observed during troubleshooting.

Example:
"Blown fuse F2 or internal power supply fault after verifying external power source and connections."

Resolution

What action was taken.

Example:
"Verified outlet and cord, allowed cooling, checked fuse F2. Error persisted; removed from service and sent for repair."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Power-related issues should always be approached from external checks first. Confirming simple factors like power source and connections helps ensure safe and efficient troubleshooting while avoiding unnecessary escalation.

That is successful troubleshooting.
