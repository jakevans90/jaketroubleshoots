GE Healthcare Aespire

Flow Sensor Calibration Fault

Asset Type

Anesthesia Machine

Manufacturer

GE Healthcare

Model

Aespire

What This Guide Helps With

Identifies causes of inaccurate flow readings or calibration errors, including sensor misalignment, obstruction, or software issues, before patient exposure.

Step-by-Step Troubleshooting

Ensure Patient Safety

  • Confirm no patient is connected to the machine.
  • Move the patient to an alternate anesthesia workstation if ventilation is required.

Check Power and Startup

  • Verify the machine powers on without errors.
  • Ensure the battery is charged or the device is connected to a stable outlet.
  • Expected outcome: No startup errors, normal boot sequence.

Inspect Flow Sensor Connections

  • Check that all gas sampling lines, flow sensor connectors, and circuit tubing are securely connected.
  • Look for kinks, disconnections, or moisture in the lines.
  • Expected outcome: Proper connection, no visible damage.

Visual Inspection of Flow Sensor

  • Inspect the sensor for physical damage, contamination, or moisture.
  • Clean or dry the sensor if needed using manufacturer-approved methods.
  • Expected outcome: Sensor is clean and unobstructed.

Run Machine Self-Test / Calibration

  • Access the Aespire calibration menu.
  • Initiate automatic flow sensor calibration following on-screen prompts.
  • Observe for error messages or abnormal readings.
  • Expected outcome: Calibration completes successfully without error.

Check Gas Supply and Pressure

  • Ensure all required gases are available and pressures are within normal ranges.
  • Low or fluctuating gas pressure can trigger calibration errors.
  • Expected outcome: Gas pressures stable, no error alarms.

Swap Components if Available

  • If another compatible flow sensor or circuit is available, swap to test whether the error persists.
  • Expected outcome: If error resolves, original sensor may need repair or replacement.

If the Problem Persists

External causes have been ruled out.

Remove the machine from service.

Label it Out of Service.

Send for GE Healthcare repair or bench evaluation.

Knowing when to escalate is proper troubleshooting.

Clinical Use Tip

Never troubleshoot the flow sensor while a patient is connected.

Always ensure ventilation is provided with an alternate machine before proceeding.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Flow sensor calibration fails during pre-use checkout."

Cause

What was observed during troubleshooting.

Example:
"Sensor obstructed by moisture or connector misaligned; self-test fails."

Resolution

What action was taken.

Example:
"Checked connections, cleaned sensor, attempted calibration; issue persists, machine removed from service and labeled Out of Service."

Helpful Details to Include (If Known)

  • Gas supply pressures verified
  • Tubing and connectors inspected
  • Self-test results recorded
  • Sensor cleanliness and dryness
  • Final device status: Out of Service

Final Thought

Patient safety and proper ventilation take priority. Systematic troubleshooting identifies external issues while avoiding unnecessary internal disassembly. Escalation to repair ensures the machine is safe and reliable. Document all findings clearly.

That is successful troubleshooting.
