Kangaroo OMNI Feeding Pump

Set Not Recognized

Asset Type

Feeding Pump

Manufacturer

Cardinal Health

Model

Kangaroo OMNI

What This Guide Helps With

This guide provides a structured Clinical Engineering approach when a Kangaroo Omni feeding pump displays a “Set Not Recognized” message.

The focus is on simple, external, and easily verifiable causes—such as disposable set compatibility, cassette seating, and contamination—before assuming internal sensor failure.

Step-by-Step Troubleshooting

1. Ensure Patient Safety

  • If feeding is active, transition the patient to another pump before troubleshooting.
  • Do not troubleshoot while connected to a patient.

2. Verify Correct Feeding Set

  • Confirm the set is approved for Kangaroo Omni use.
  • Verify the set is not expired.
  • Inspect the cassette for cracks, warping, or visible damage.
  • Incompatible or damaged sets commonly trigger this error.

3. Remove and Reinstall the Feeding Set

  • Fully remove the cassette from the pump.
  • Reinsert it into the molded receptacle.
  • Ensure it sits flush and fully engages the retention mechanism.
  • Improper alignment prevents the internal sensor from recognizing the set.

4. Inspect the Cassette Interface

  • Check for dried formula or debris inside the receptacle.
  • Clean gently with an approved disinfectant wipe.
  • Ensure no moisture enters the housing.
  • Contamination can interfere with cassette detection.

5. Power Cycle the Pump

  • Turn the pump off.
  • Disconnect AC power.
  • Wait 30 seconds.
  • Restart the pump and retest.

6. Install a New Feeding Set

  • Open a new compatible set.
  • Install carefully and confirm proper seating.
  • If the pump recognizes the new set, the original set was likely defective.

If the Problem Persists

If a new compatible set does not resolve the issue and the cassette area is clean and properly aligned, the issue is likely internal (sensor failure or control board issue).

  • Remove the pump from service.
  • Label it Out of Service.
  • Send for bench evaluation or vendor repair.

Knowing when to stop is proper troubleshooting.

Clinical Use Tip

Always secure an alternate feeding pump before beginning troubleshooting. Even short interruptions in enteral therapy can impact critically ill patients.

Work Order Documentation (CCR Method)

Complaint

Explain what was reported.

Example:
“Pump displays ‘Set Not Recognized’ when attempting to start feeding.”

Cause

Explain what was observed or determined.

Example:
Disposable feeding set not fully seated in cassette receptacle.

OR

Defective disposable set.

OR

Suspected internal cassette recognition sensor failure after external checks completed.

Resolution

Explain what action was taken.

Example:
Removed and reinstalled feeding set. Verified proper seating and normal operation.

OR

Replaced disposable set with new compatible set. Pump functioning normally.

OR

Removed pump from service and sent for repair for suspected internal fault.

Helpful Details to Include (If Known)

  • Feeding set type
  • Lot number and expiration date
  • Whether new set resolved issue
  • Cassette alignment observation
  • Debris found in receptacle
  • Power cycle performed
  • Final device status

Final Thought

Structured, external-first troubleshooting prevents unnecessary repairs and reduces downtime. Most set recognition issues are related to disposable set installation or compatibility. Prioritize patient safety, verify simple causes first, and escalate appropriately when internal failure is suspected.

That is successful troubleshooting.
