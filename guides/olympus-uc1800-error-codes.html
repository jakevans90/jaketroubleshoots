Olympus UC1800 Ultrasonic Cleaner

Error Codes / Sensor Faults

Asset Type

Ultrasonic Cleaner

Manufacturer

Olympus

Model

UC1800

What This Guide Helps With

This guide addresses situations where an Olympus UC1800 ultrasonic cleaner displays error codes or sensor fault messages. The focus is on external and easily verifiable causes—such as water level, power, connections, and cleaning solution—before assuming internal sensor or board failure. The goal is to restore safe operation while ensuring proper documentation and escalation if needed.

Step-by-Step Troubleshooting

Power Check

  • Verify the unit is plugged into a working outlet.
  • Check for blown fuses or tripped breakers.
  • Ensure the power switch is ON.

Why: No power or unstable power can cause false sensor errors.

Water Level and Quality

  • Ensure the tank is filled to the minimum and maximum levels.
  • Confirm water is clean; overly dirty solution can trigger error codes.

Why: The UC1800 has sensors that detect fluid presence and conductivity.

Temperature Sensor Check

  • If equipped, confirm the heater sensor is fully immersed in water.
  • Look for bent or disconnected probes.

Why: Dry operation or sensor misalignment can trigger error codes.

Connection and Cable Inspection

  • Inspect all internal and external connectors for corrosion, moisture, or looseness.
  • Verify that any removable sensor connectors are seated properly.

Why: Loose or dirty connections are a common cause of sensor faults.

Cleaning Cycle Reset

  • Power cycle the unit (turn OFF → wait 30 seconds → turn ON).
  • Try starting a basic cleaning cycle without ultrasonic power (if possible).

Why: Software glitches may clear with a simple reset.

Solution Conductivity

  • Ensure the cleaning solution is compatible with the UC1800.
  • Avoid using pure water without proper ultrasonic detergent.

Why: Incorrect solution conductivity may trigger sensor fault codes.

If the Problem Persists

External checks have been ruled out.

Device may have internal sensor or circuit board failure.

  • Remove the UC1800 from service.
  • Label as Out of Service.
  • Send for bench evaluation or manufacturer repair.

Note: Continuing operation without proper function can damage the unit and compromise cleaning efficacy.

Clinical Use Tip

  • Do not run the ultrasonic cleaner empty; always ensure proper water level.
  • Move instruments to an alternate cleaner if this unit is unavailable to maintain workflow.
  • Always verify patient instruments are properly cleaned using a functioning unit.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
“UC1800 displays sensor error E02 and will not start cleaning cycle.”

Cause

What was observed during troubleshooting.

Example:
Observed water level below minimum; heater sensor partially lifted out of solution.

Resolution

What action was taken.

Examples:
Refilled tank to proper level and ensured sensor fully immersed; cycled power and verified normal operation. If issue persists, unit removed for repair.

Helpful Details to Include

  • Water level at time of error
  • Cleaning solution type and age
  • Tank temperature at error
  • Error code displayed (e.g., E01, E02)
  • Power supply voltage and outlet check results
  • Final device status (working, out of service, sent for repair)

Final Thought

Proper ultrasonic cleaner function is critical for instrument decontamination. By following a logical, external-first troubleshooting approach, patient safety is preserved, unnecessary repairs are avoided, and documentation supports future maintenance decisions. Always escalate when internal faults are suspected rather than attempting deep repair.

That is successful troubleshooting.
