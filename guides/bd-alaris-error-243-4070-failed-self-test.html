BD Alaris

Error: 243.4070 Failed self test

Asset Type

Infusion Pump

Manufacturer

BD

Model

Alaris

What This Guide Helps With

Error 243.4070 indicating air-in-line subsystem self-test failure, preventing infusion due to inability to verify air detection safety function.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Remove the pump from active patient use immediately
  • Transfer infusion to a functioning pump
  • Clamp tubing as needed to maintain safe therapy transition

Verify the Reported Error and Behavior

  • Power on the pump and observe startup sequence
  • Confirm Error 243.4070 appears during self-test
  • Note if the pump fails to proceed to normal operation
  • Check if the error occurs consistently at startup

Check for Set Loading and Channel Setup Issues

  • Ensure correct BD-compatible administration set is used
  • Open and reseat the set in the pump channel
  • Verify tubing is properly aligned in the air-in-line sensor path
  • Ensure the door is fully latched

Inspect Air-in-Line Sensor Area

  • Open the pump door and inspect the detection channel
  • Look for fluid contamination, residue, or debris
  • Check for dried medication buildup in the sensor path
  • Clean per facility-approved methods if contamination is present

Power Cycle the Pump

  • Turn the pump or channel off
  • Disconnect from AC power if applicable
  • Wait 10–15 seconds
  • Restart and observe the self-test

Test with No Set Installed

  • Power on the pump with the channel empty
  • Observe if the error still occurs

Test with a Known-Good Set

  • Install a new, known-good BD-compatible administration set
  • Load carefully and restart the pump

If the Problem Persists

If Error 243.4070 continues after verifying setup, cleaning the sensor area, power cycling, and testing with and without a set, external causes have been ruled out.

  • Remove the device from service
  • Label Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot on an active patient. Always move the patient to a backup device before attempting any troubleshooting steps.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"BD Alaris pump displaying Error 243.4070 during startup, preventing infusion"

Cause

What was observed during troubleshooting.

Example:
"Air-in-line subsystem failed internal self-test; error persisted after reseating set, cleaning sensor, and testing with known-good tubing"

Resolution

What action was taken.

Example:
"Removed pump from service, labeled out of service, and sent for repair"

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Self-test failures involving air detection are high-risk conditions. Proper troubleshooting focuses on ruling out setup and contamination issues quickly, then escalating appropriately to protect patient safety.

That is successful troubleshooting.
