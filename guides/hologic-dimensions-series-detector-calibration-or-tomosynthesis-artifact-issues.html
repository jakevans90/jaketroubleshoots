Hologic Dimensions Series

Detector Calibration or Tomosynthesis Artifact Issues

Asset Type

Mammography System

Manufacturer

Hologic

Model

Dimensions Series

What This Guide Helps With

Identifies external causes of detector artifacts or tomo image issues, including contamination, calibration state, positioning, or environmental factors affecting image quality.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Confirm no patient is in compression and no exposure is active.
  • If needed, move the patient to another unit.

Expected Outcome:
Safe environment for troubleshooting with no radiation risk.

Identify Artifact Type and Pattern

  • Review recent images and determine:
  • Lines, bands, or pixel dropouts
  • Shading or brightness non-uniformity
  • Artifacts only in tomosynthesis vs 2D
  • Fixed vs intermittent appearance

Expected Outcome:
Pattern helps determine if issue is calibration, contamination, or system-related.

Why: Artifact behavior is key to narrowing the source quickly.

Verify Artifact Across Displays and Systems

  • Check images on:
  • Acquisition workstation
  • Review workstation (if available)
  • PACS

Expected Outcome:
Artifact only on one display → monitor issue
Artifact everywhere → detector/system issue

Why: Confirms whether the issue is image data vs display-related.

Inspect Detector Surface and Compression Paddle

  • Check for:
  • Dust, debris, or contrast residue
  • Fingerprints or fluid contamination
  • Damaged or dirty compression paddles
  • Clean using approved wipes if needed.

Expected Outcome:
Removal of contamination eliminates image artifacts if surface-related.

Why: Even minor debris can create visible artifacts in mammography imaging.

Verify Paddle Type and Positioning

  • Ensure:
  • Correct paddle type is installed
  • Paddle is properly seated and aligned
  • No tilt or uneven compression surface

Expected Outcome:
Proper positioning removes geometric or shading artifacts.

Why: Misalignment affects beam geometry and detector response.

Check Calibration Status

  • Review system for:
  • Calibration due/expired alerts
  • Recent calibration failures
  • Flat-field or offset calibration status
  • Run calibration if appropriate and allowed.

Expected Outcome:
Successful calibration resolves fixed pattern or shading artifacts.

Why: Detector calibration corrects pixel sensitivity variations.

Evaluate Environmental Factors

  • Check for:
  • Temperature or humidity outside normal range
  • Recent system warm-up or startup
  • Nearby equipment causing vibration or interference

Expected Outcome:
Stable environment improves detector consistency.

Why: Detector performance can drift with environmental instability.

Perform System Restart

  • Shut down using proper procedure, wait 2–3 minutes, then restart.

Expected Outcome:
Temporary processing or calibration glitches may clear.

Why: Resets detector communication and processing pipelines.

Check for Recent Changes or Service Activity

  • Determine if:
  • Detector was recently serviced or replaced
  • Software updates were performed
  • Calibration was interrupted previously

Expected Outcome:
Identifying recent changes helps correlate cause of artifact.

Why: Many artifact issues follow incomplete setup or service events.

If the Problem Persists

If artifacts remain after cleaning, calibration, and environmental checks, common external causes have been ruled out.

The issue is likely internal (detector panel, calibration hardware, or processing system).

Remove the system from service
Label as Out of Service
Escalate for vendor service or advanced bench evaluation

Knowing when to stop prevents unnecessary risk and delays.

Clinical Use Tip

Do not troubleshoot imaging artifacts during an active patient exam. Always move the patient to another system before evaluating image quality issues.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Staff reported persistent vertical line artifacts on tomosynthesis images."

Cause

What was observed during troubleshooting.

Example:
"Contamination and incomplete detector calibration identified during inspection."

Resolution

What action was taken.

Example:
"Detector surface cleaned and calibration performed; artifacts resolved and system returned to service."

Helpful Details to Include (If Known)

  • Artifact type (lines, bands, shading)
  • Appears in 2D, tomo, or both
  • Present on all workstations/PACS
  • Detector cleaning performed
  • Calibration status/results
  • Environmental conditions
  • Any recent service or updates
  • Final system status

Final Thought

Artifact troubleshooting should always start with simple, external causes like contamination, positioning, and calibration. Maintaining patient safety, following a logical process, and escalating appropriately ensures reliable imaging and avoids unnecessary downtime.

That is successful troubleshooting.
