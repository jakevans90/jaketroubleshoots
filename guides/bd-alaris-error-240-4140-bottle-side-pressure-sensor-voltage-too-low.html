BD Alaris

Error: 240.4140 Bottle-Side Pressure Sensor Voltage Too Low

Asset Type

Infusion Pump

Manufacturer

BD

Model

Alaris

What This Guide Helps With

This guide addresses Error 240.4140 Bottle-Side Pressure Sensor Voltage Too Low on the BD Alaris infusion pump. This condition indicates the system cannot properly read upstream pressure, which can impact occlusion detection and may stop or prevent infusion.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Remove the pump from patient use if infusion accuracy is in question
  • Transfer infusion to another functioning pump

Verify the Reported Error and Behavior

  • Power on the pump and confirm Error 240.4140 is displayed
  • Observe when the error occurs (startup, setup, or during infusion)
  • Check if infusion is stopped or prevented
  • Note any additional pressure or occlusion-related alarms

Check Administration Set and Loading

  • Open the pump door and inspect the administration set
  • Confirm use of a correct BD-approved Alaris administration set
  • Ensure tubing is properly routed through the pump channel
  • Verify no kinks, pinching, or twisting of tubing
  • Reseat the tubing and close the door securely

Inspect for Contamination or Fluid Intrusion

  • Inspect the pump channel and pressure sensing area
  • Look for fluid spills, residue, or dried medication
  • Check for moisture inside the pumping channel
  • Clean per facility protocol if contamination is present
  • Allow area to fully dry before retesting

Check Door and Latch Operation

  • Verify the door fully latches without resistance
  • Check for looseness, misalignment, or damage
  • Ensure proper compression of tubing when closed

Power Cycle the Pump

  • Turn the pump off and wait 10–15 seconds
  • Power the pump back on and observe for error clearance

Test with Known-Good Setup

  • Install a new, known-good administration set
  • Repeat setup process
  • Attempt to start infusion if safe to test
  • Monitor for recurrence of the error

If the Problem Persists

If Error 240.4140 continues after all external checks, external causes have been ruled out. This strongly indicates an internal pressure sensor or analog circuit failure.

  • Remove device from service
  • Label Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot on an active patient. Move the patient to a backup device first before performing any troubleshooting steps.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"BD Alaris pump displaying Error 240.4140 (Bottle-Side Pressure Sensor Voltage Too Low), preventing infusion."

Cause

What was observed during troubleshooting.

Example:
"Error persisted after verifying proper set installation, clean pump channel, and normal external conditions, indicating internal sensor fault."

Resolution

What action was taken.

Example:
"Removed pump from service, labeled out of service, and sent for repair evaluation."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Pressure sensor errors on BD Alaris pumps often indicate true hardware faults once setup issues and contamination are ruled out. Proper escalation protects patient safety and ensures accurate infusion delivery.

That is successful troubleshooting.
