Medivators DSD-201 / DSD Edge Series

Leak Test Failure Preventing Cycle Initiation

Asset Type

Endoscope Reprocessor (AER)

Manufacturer

Medivators

Model

DSD-201 / DSD Edge Series

What This Guide Helps With

Leak test failures that prevent cycle start due to scope connection issues, damaged channels, or improper setup.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Ensure no scope in use is needed for an active procedure. Remove the scope from clinical use before troubleshooting.
  • Expected: No patient impact
  • Why it matters: Prevents delays in care and ensures safe handling of contaminated scopes

Verify the Reported Error and Behavior

  • Attempt to start a cycle and observe the leak test failure message.
  • Confirm leak test fails immediately or during pressurization
  • Note any error codes or messages
  • Expected: Leak test failure is reproducible
  • Why it matters: Confirms true system-detected leak vs user interruption

Check Scope Connection to AER (Most Common Cause)

  • Inspect how the endoscope is connected to the AER ports.
  • Ensure all channel connectors are properly attached
  • Verify correct hookup for the specific scope type
  • Disconnect and reconnect all fittings securely
  • Expected: Tight, correct connections with no looseness
  • Why it matters: Loose or incorrect connections will fail leak integrity checks

Inspect Leak Test Tubing and Adapters

  • Check all external tubing used for leak testing.
  • Inspect tubing for cracks, kinks, or disconnections
  • Verify adapters are correct and not damaged
  • Swap with known good tubing/adapters if available
  • Expected: Tubing is intact and properly fitted
  • Why it matters: External leaks in tubing will cause false leak test failures

Check the Endoscope for External Damage or Moisture

  • Visually inspect the scope prior to testing.
  • Look for visible damage, cracks, or worn insertion tube
  • Check for fluid intrusion or residual moisture
  • Ensure scope was properly pre-cleaned and dried
  • Expected: Scope appears intact and dry
  • Why it matters: Actual scope leaks will correctly fail the leak test

Verify Proper Channel Hookup Configuration

  • Ensure all required channels are connected correctly.
  • Confirm all channels required for the model scope are connected
  • Ensure unused ports are properly capped if required
  • Expected: Correct channel configuration
  • Why it matters: Missing or incorrect channel connections can prevent pressure stabilization

Check Door Seal and Latch Integrity

  • Inspect the AER door and sealing surface.
  • Ensure door is fully closed and latched
  • Inspect gasket for damage, debris, or wear
  • Clean gasket if contaminated
  • Expected: Door seals completely with no gaps
  • Why it matters: Air leaks from the chamber can trigger leak test failure

Restart the AER System

  • Perform a system reset.
  • Power cycle the unit
  • Restart and attempt leak test again
  • Expected: System reinitializes and may clear transient faults
  • Why it matters: Software or sensor faults can cause false failures

If the Problem Persists

If all external connections, tubing, scope condition, and setup have been verified, the issue is likely internal (pressure sensor, internal plumbing, or control system fault).

Remove the device from service, label it Out of Service, and send for repair or further bench evaluation.

Knowing when to stop is proper troubleshooting.

Clinical Use Tip

Do not troubleshoot leak test failures on a scope intended for immediate patient use. Always move to a backup scope to avoid procedural delays.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"AER cycle would not start due to repeated leak test failure."

Cause

What was observed during troubleshooting.

Example:
"Loose channel connection at scope interface causing pressure loss during leak test."

Resolution

What action was taken.

Example:
"Reconnected all scope channels securely and verified successful leak test and cycle initiation."

Helpful Details to Include (If Known)

  • Leak test behavior (immediate vs gradual failure)
  • Scope type and channel configuration used
  • Tubing/adapters condition and swaps
  • Door seal condition
  • Any visible scope damage or moisture
  • Final device status (passed leak test or removed from service)

Final Thought

Leak test failures are most often caused by external setup issues, not internal faults. A methodical check of connections, tubing, and scope condition will resolve most cases. Escalate appropriately when integrity cannot be confirmed, and always prioritize patient safety and proper documentation.

That is successful troubleshooting.
