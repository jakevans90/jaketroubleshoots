CONMED System 2450 / 5000 Series

Error: Err 383 RF current sensed at an unselected output

Asset Type

Electrosurgical Unit (ESU)

Manufacturer

CONMED

Model

System 2450 / 5000 Series

What This Guide Helps With

This guide helps troubleshoot Err 383 on the CONMED System 2450 / 5000 Series electrosurgical unit when RF current is sensed at an output that was not selected. This condition may indicate an accessory issue, contaminated output connector, incorrect setup, faulty output switching, output sensing problem, stuck relay, Output assembly fault, or Controller assembly fault.

Because this error involves possible RF current on the wrong output path, the device should be treated as a patient and staff safety concern until proper testing confirms safe operation.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Stop use of the generator immediately if Err 383 appears during setup, activation, testing, or clinical use.
  • Do not continue using the ESU on a patient with this error active or recurring.
  • Have clinical staff switch to a known working backup ESU if electrosurgery is still needed.
  • Disconnect the active electrode, return electrode cable, bipolar accessories, adapters, handpieces, and footswitches.
  • Label the unit for Clinical Engineering evaluation if the error repeats or cannot be cleared safely.

Verify the Reported Error and Behavior

  • Confirm the exact displayed error is Err 383.
  • Ask staff when the error occurred: at startup, during activation, in a specific mode, or intermittently during use.
  • Document which output was selected when the error appeared.
  • Document whether the issue occurred during monopolar cut, monopolar coag, bipolar use, or another output setup.
  • Power the unit on only in a safe non-patient-use setting to confirm whether the error repeats.

Inspect the External Setup First

  • Check the ESU power cord, wall outlet, equipotential grounding connection if used, and general condition of the unit.
  • Inspect all output connectors for bent contacts, contamination, fluid intrusion, burn marks, loose connectors, or foreign material.
  • Look for carbon tracking, residue, or signs of arcing around the front panel output receptacles.
  • Do not return the unit to service if any output connector appears damaged, contaminated, burned, or unsafe.

Remove Accessories and Retest

  • Disconnect all accessories from all outputs.
  • Remove active pencils, bipolar forceps, adapters, footswitches, return electrode cables, and specialty accessories.
  • Cycle power and check whether Err 383 returns with nothing connected.
  • If the error clears, reconnect accessories one at a time using proper ports and retest.
  • If the error follows a specific accessory or cable, remove that accessory from service and retest the generator with known good accessories.

Check for Accessory or Cable Cross-Connection Issues

  • Verify accessories are connected to the correct output ports.
  • Confirm monopolar and bipolar accessories are not being used through incorrect adapters.
  • Inspect cables for cuts, crushed insulation, exposed conductors, melted areas, or signs of arcing.
  • Swap suspect accessories with known good accessories if available.
  • Confirm the error does not follow a specific accessory, cable, adapter, or handpiece.

Verify Output Selection and Activation Method

  • Confirm the selected mode and output match the accessory being tested.
  • Test activation using the proper hand switch or footswitch in a controlled bench setting.
  • Check whether the error occurs only with hand activation, footswitch activation, a specific mode, a specific power setting, or a specific output connector.
  • Document whether the error is output-specific or occurs regardless of activation method.

Bench Test with Fixed Loads

  • Remove the unit from clinical use before bench testing.
  • Use appropriate ESU test equipment and fixed loads.
  • Connect fixed loads to the outputs as required for controlled testing.
  • Activate each output one at a time.
  • Monitor all outputs for RF current while only one output is selected and activated.
  • RF current should only be detected at the selected output.

Identify Whether the Fault Is Output-Specific

  • Document which selected output causes the error.
  • Document which unselected output shows current or triggers the fault.
  • Test each output individually under controlled conditions.
  • Compare behavior across monopolar and bipolar outputs.
  • A fault that follows one output path may indicate a stuck or faulty output relay.
  • A fault that appears across multiple paths may suggest an output sensing, Output assembly, or Controller assembly issue.

Stop When Resolved

  • If the issue is traced to an external accessory, damaged cable, contaminated connector, or incorrect setup, remove the faulty item from service.
  • Clean or correct the external issue only if appropriate and allowed by facility procedure.
  • Retest the ESU with known good accessories and proper loads.
  • Verify normal operation before return to service.
  • Complete required performance and safety checks per facility procedure.

If the Problem Persists

If Err 383 continues after accessories, cables, setup, connectors, activation method, and fixed-load testing have been checked, external and common causes have been ruled out.

Persistent Err 383 suggests an internal fault involving output switching, output sensing, a stuck or faulty output relay, the Output assembly, or the Controller assembly.

The device should be removed from service, labeled Out of Service, and sent for repair or bench evaluation. Do not return the ESU to patient use until output switching and safety performance have been verified.

Clinical Use Tip

Never troubleshoot this error on an active patient or during a procedure. Err 383 involves possible RF current on the wrong output path. Move the patient or case to a known working backup ESU first, then evaluate the affected generator outside the patient care area.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"CONMED System 2450 / 5000 Series ESU reported Err 383. Staff reported RF current sensed at an unselected output during activation/testing."

Cause

What was observed during troubleshooting.

Example:
"Verified Err 383 condition. External accessories, output cables, activation method, and connector condition checked. Fixed-load testing indicated possible RF current detection on an output that was not selected."

Resolution

What action was taken.

Example:
"Removed ESU from service and labeled Out of Service. Unit requires repair or further bench evaluation of output switching, output sensing, Output assembly, and Controller assembly before return to service."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Err 383 should be treated as a serious ESU output control fault until proven otherwise. Start with patient safety, remove external causes first, and use controlled fixed-load testing to determine whether current is appearing on an unselected output. If the fault persists, escalation is appropriate because output relay, Output assembly, or Controller assembly failure may be involved.

That is successful troubleshooting.
