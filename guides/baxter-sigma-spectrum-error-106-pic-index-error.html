Baxter SIGMA Spectrum

Error: 106 PIC Index Error

Asset Type

Infusion Pump

Manufacturer

Baxter

Model

SIGMA Spectrum

What This Guide Helps With

Error 106 PIC Index Error indicates the pump cannot track motor position, leading to infusion interruption, inability to start, or inaccurate delivery.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Remove the pump from active patient use immediately.
  • Transfer infusion to another functioning pump.

Verify the Reported Error and Behavior

  • Power on the pump and confirm Error 106 PIC Index Error is displayed.
  • Observe when the error occurs (startup, during infusion, or after loading a set).
  • Check for pump not running, abnormal motor sounds, or failure to initiate infusion.

Check for Cassette or Set Loading Issues

  • Open the pump door and inspect the IV set or cassette.
  • Ensure a correct Baxter-approved set is installed.
  • Verify proper seating and alignment.
  • Look for kinks, resistance, or misalignment.
  • Reload the set carefully and close the door fully.

Inspect for Obstructions in Pump Mechanism

  • With the door open, visually inspect for debris, dried fluid, or foreign objects.
  • Check the pumping channel and visible moving components.
  • Do not insert tools—only clear obvious debris safely.

Power Cycle the Pump

  • Turn the pump off completely.
  • Wait 10–15 seconds and power it back on.

Test with a Known-Good Set

  • Install a different known-good IV set.
  • Attempt to start infusion.

Observe for Recurrence

  • Monitor for immediate or intermittent return of the error.
  • Verify whether infusion can run consistently.

If the Problem Persists

If Error 106 continues after set replacement, inspection, and power cycling, external causes have been ruled out.

Remove the device from service, label it Out of Service, and send for repair or bench evaluation.

Clinical Use Tip

Never troubleshoot on an active patient. Always transfer the patient to a backup device before continuing diagnostics.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Pump displaying Error 106 PIC Index Error; infusion would not start."

Cause

What was observed during troubleshooting.

Example:
"Internal motor/index tracking failure suspected after verifying proper set loading and replacing IV set with no resolution."

Resolution

What action was taken.

Example:
"Removed pump from service, labeled Out of Service, and sent for repair evaluation."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Index errors indicate loss of positional control within the pump’s drive system. Always rule out loading and obstruction first, but escalate quickly when errors persist to maintain patient safety and device reliability.

That is successful troubleshooting.
