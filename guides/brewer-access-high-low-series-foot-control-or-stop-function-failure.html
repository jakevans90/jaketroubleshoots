Brewer Access High-Low Series

Foot Control or Stop-Function Failure

Asset Type

Exam Table

Manufacturer

Brewer

Model

Access High-Low Series

What This Guide Helps With

Troubleshooting foot control response, Return-to-Chair interruption, or stop-function failure before suspected control, switch, wiring, or actuator repair.

Step-by-Step Troubleshooting

1. Ensure Patient Safety First

Remove the table from patient use before testing any powered movement or stop function.

If a patient is on the table and movement is unreliable, assist the patient off the table using appropriate clinical support before troubleshooting.

Expected outcome: The table is unoccupied and can be safely operated through its range of motion without patient risk.

Why it matters: A foot control or stop-function failure may prevent reliable positioning or interruption of automatic motion.

2. Confirm the Reported Failure

With the table unoccupied, identify the exact complaint:

  • No movement from one or more foot control pedals
  • Intermittent foot control operation
  • Return-to-Chair function does not activate
  • Red Stop button does not stop automatic Return-to-Chair movement
  • A foot control pedal does not interrupt automatic Return-to-Chair movement

For models equipped with powered back and Return-to-Chair operation, Brewer specifies that pressing the red Stop button or any foot control pedal during Return-to-Chair motion should stop the movement and leave the table stopped.

Expected outcome: The failure can be reproduced and documented accurately.

If the issue cannot be reproduced: Complete a functional safety check, document findings, and return the table to service only when all installed controls operate correctly.

3. Inspect the Foot Control Externally

Inspect the foot control for:

  • Cracked housing
  • Damaged, stuck, or missing pedals/buttons
  • Fluid contamination or debris around the pedals
  • Pinched, cut, frayed, or stretched cord
  • Signs the cable was pulled, rolled over, or caught under the table

Depress each pedal and button by hand. Confirm that each returns freely without sticking.

Expected outcome: The foot control and cable are physically intact, and each control moves freely.

If damage or contamination is found: Remove the table from service and replace or evaluate the affected foot control assembly before further clinical use.

4. Check Power and Main Table Operation

Verify that:

  • The power cord is securely connected.
  • The outlet is functional.
  • The table power switch is in the ON position.
  • The table shows normal powered operation when commanded.

Using only the controls present on the table, test basic table-up, table-down, back-up, and back-down movement as applicable.

Expected outcome: The table has reliable power and at least identifies whether the failure affects all powered movement or only specific foot control functions.

If all powered movement is absent: The problem may involve power, control box, connection, or protective circuitry rather than only the foot control. Remove from service for bench evaluation.

5. Check the Foot Control Cable Connection

Inspect the external foot control connection at the rear or control interface area of the table, as equipped.

Verify that the connector is:

  • Fully seated
  • Properly aligned
  • Not loose
  • Free of bent pins, broken locking features, contamination, or visible damage

Disconnect and reconnect the foot control only when the table is powered off and it is safe to do so.

Expected outcome: A securely seated connection restores reliable foot control response, or confirms that a loose external connection is not the cause.

If operation is restored: Perform the complete functional test below before returning the table to service.

6. Test Individual Foot Control Functions

With the table unoccupied and clear of obstructions, operate each available foot control function separately:

  • TABLE UP
  • TABLE DOWN
  • BACK UP, if equipped
  • BACK DOWN, if equipped
  • RETURN TO CHAIR, if equipped
  • STOP, if equipped

Observe whether the related function moves normally, hesitates, activates intermittently, or does not respond.

Expected outcome: Each installed control produces the intended movement consistently without unexpected motion, abnormal sound, or continued motor operation after reaching its limit.

If only one pedal or button fails while other functions operate: Suspect a failed foot control input or related control circuit. Remove the table from service for repair evaluation.

7. Test the Stop Function During Return-to-Chair Motion

This step applies only to models equipped with powered backrest and Return-to-Chair/Stop controls.

Position the unoccupied table so Return-to-Chair movement can occur safely. Press the Return-to-Chair button, then momentarily press the red Stop button while the table is moving.

Expected outcome: The table immediately stops automatic movement and remains stopped. Brewer identifies this as the required operating test for the stop button.

If the table does not stop: Immediately discontinue testing, turn the table off if necessary to prevent unsafe motion, and remove it from service.

8. Test Pedal Interruption of Automatic Motion

For models equipped with Return-to-Chair operation, repeat the automatic motion test and momentarily press each available motion pedal during movement:

  • BACK UP, if equipped
  • BACK DOWN, if equipped
  • TABLE UP
  • TABLE DOWN

Expected outcome: Pressing any pedal during automatic Return-to-Chair movement stops the table and leaves it stopped, as specified in Brewer’s operating test.

If any pedal fails to interrupt movement: The table does not pass its stop-function safety check and must not be returned to clinical use.

9. Observe for Intermittent or Unsafe Behavior

During testing, observe for:

  • Movement continuing after the pedal is released
  • Return-to-Chair starting but not stopping correctly
  • Unexpected direction of movement
  • Delayed response
  • Motor hum without movement
  • Abnormal noise
  • Unusual heat or odor
  • Movement stopping only when power is removed

Expected outcome: Table movement occurs only when intended and stops reliably through all installed controls.

If unsafe or intermittent behavior occurs: Stop troubleshooting and remove the table from service for repair or bench evaluation.

If the Problem Persists

If power, external connections, visible foot control condition, individual pedal operation, and stop-function response have been checked without resolving the issue, the common external causes have been ruled out.

A failed foot control, return-to-chair control path, safety switch, control box, wiring connection, or actuator-related input may be present. Brewer’s troubleshooting documentation directs replacement or technical service escalation when Return-to-Chair motion is not stopped by the Stop button or foot control pedals.

The device should be:

  • Removed from service
  • Labeled Out of Service
  • Sent for repair or bench evaluation

Knowing when to stop troubleshooting and escalate a failed motion-safety function is proper Clinical Engineering practice.

Clinical Use Tip

Do not troubleshoot powered positioning or stop-function failures with a patient on the table. Move the patient to another safe exam surface first, then test the Brewer table only when unoccupied and clear of obstructions.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Clinical staff reported that the Brewer Access High-Low exam table would enter Return-to-Chair motion, but the red Stop button did not stop movement."

Cause

What was observed during troubleshooting.

Example:
"During functional testing with the table unoccupied, Return-to-Chair activated normally, but the Stop button and one motion pedal failed to interrupt automatic travel."

Resolution

What action was taken.

Example:
"Removed the exam table from service, labeled it Out of Service, and referred it for bench evaluation of the foot control and stop-function circuit before return to clinical use."

Helpful Details to Include (If Known)

  • Outlet tested and confirmed functional
  • Table power switch position
  • Power cord and external connection condition
  • Foot control cable condition
  • Pedals/buttons tested
  • Whether Return-to-Chair activated
  • Whether the red Stop button stopped motion
  • Whether each pedal interrupted automatic motion
  • Any unexpected movement, motor hum, noise, heat, or odor
  • Final device status

Final Thought

Powered table movement must be predictable and stoppable before the equipment is used with a patient. Logical external checks can identify simple failures, but an unreliable stop function requires prompt removal from service, appropriate escalation, and clear CCR documentation.

That is successful troubleshooting.
