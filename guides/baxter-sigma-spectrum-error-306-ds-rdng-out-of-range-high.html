Baxter SIGMA Spectrum

Error: 306 DS rdng out of range (high)

Asset Type

Infusion Pump

Manufacturer

Baxter

Model

SIGMA Spectrum

What This Guide Helps With

Error 306 indicates a downstream (DS) reading out of range (high), which can stop infusion due to abnormal pressure conditions, occlusions, or sensor-related issues affecting safe fluid delivery.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Stop the infusion if active
  • Disconnect the pump from the patient
  • Transfer therapy to a functioning infusion device

Verify the Reported Error and Behavior

  • Power on the pump and confirm Error 306 is displayed
  • Observe when the error occurs:
    • Startup
    • After loading a set
    • During infusion
  • Check for:
    • Infusion not starting
    • Repeated alarms
    • No fluid movement

Check IV Set Installation and Compatibility

  • Ensure a Baxter-approved IV set is installed
  • Verify proper routing through the pump channel
  • Confirm cassette is fully seated and door is latched

Inspect for Downstream Occlusion or Restriction

  • Trace tubing from pump to patient
  • Check for:
    • Closed clamps
    • Kinked tubing
    • Blocked filters
    • Occluded catheter or access site

Check for Fluid or Contamination in Pump Channel

  • Open the pump door and inspect for:
    • Fluid spills
    • Dried medication residue
    • Debris near sensor areas
  • Clean per manufacturer-approved procedures if needed

Replace the IV Set with a Known-Good Set

  • Remove the current set
  • Install a new, verified IV set
  • Ensure proper alignment and loading

Restart the Pump (Power Cycle)

  • Turn the pump off
  • Wait several seconds
  • Power the unit back on

Attempt Operation Without Patient Connection (Test Setup)

  • Prime a new set
  • Run the pump into a test container
  • Monitor for recurrence of Error 306

If the Problem Persists

If Error 306 continues after all external checks:

  • External causes have been ruled out
  • The issue likely indicates internal sensor or pressure detection failure
  • Remove the pump from service
  • Label as Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot pressure-related alarms while connected to a patient. Always transfer infusion to a backup device before continuing diagnostics.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Pump displaying Error 306 DS rdng out of range (high); infusion would not start."

Cause

What was observed during troubleshooting.

Example:
"High downstream pressure detected. External causes ruled out; suspected internal sensor fault."

Resolution

What action was taken.

Example:
"Replaced IV set and verified tubing path. Removed pump from service after error persisted."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Error 306 indicates a downstream pressure or sensing issue. Always rule out occlusions and setup problems first before concluding a device fault. Proper troubleshooting ensures safe operation and efficient escalation when needed.

That is successful troubleshooting.
