Hologic Dimensions Series

Compression paddle sensor or force calibration errors

Asset Type

Mammography System

Manufacturer

Hologic

Model

Dimensions Series

What This Guide Helps With

Identifies external causes of paddle sensor or force calibration errors, including misalignment, incorrect paddles, obstructions, or system readiness issues.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Confirm no patient is in compression and the paddle is fully released.
  • If necessary, move the patient to another unit.

Expected Outcome: Safe environment with no risk of unintended compression.

Observe Error Message and Paddle Behavior

  • Note the exact error message.
  • Attempt small paddle movements using controls.

Expected Outcome:

  • No movement → possible interlock or recognition issue
  • Movement with error → possible calibration or sensor issue

Why: Helps determine if the issue is mechanical, sensor-related, or system logic.

Verify Compression Paddle Installation

  • Remove and reseat the compression paddle.
  • Ensure it is fully seated and locked.
  • Confirm correct paddle type for the exam.
  • Verify proper orientation.

Expected Outcome: System recognizes paddle and clears sensor error.

Why: Improper seating or incorrect paddle type is a common cause of sensor faults.

Inspect Paddle and Mounting Interface

  • Check for debris, damage, or wear on paddle.
  • Inspect for bent or damaged mounting brackets.
  • Look for obstruction at docking interface.
  • Clean or remove any visible obstruction.

Expected Outcome: Clear interface allows proper sensor detection.

Why: Physical interference can prevent accurate paddle recognition or force sensing.

Check for Obstructions in Compression Path

  • Ensure nothing is between the paddle and detector.
  • Check for pads, covers, or foreign objects.
  • Verify accessories are not misaligned.

Expected Outcome: Unobstructed motion allows normal force calibration.

Why: Unexpected resistance can trigger force calibration errors.

Verify System Ready and Calibration State

  • Check system status for readiness.
  • Look for any calibration prompts.
  • If available, attempt a user-level calibration or reset sequence.

Expected Outcome: System completes calibration or clears fault.

Why: Some errors occur if calibration was interrupted or not completed properly.

Power Cycle the System

  • Perform a full system shutdown and restart following proper procedure.

Expected Outcome: Temporary faults or sensor communication issues are cleared.

Why: Resets system logic and reinitializes paddle detection.

Test with Alternate Paddle (If Available)

  • Install a different compatible compression paddle.

Expected Outcome:

  • Issue resolved → original paddle likely faulty
  • Issue persists → system-side issue

Why: Helps isolate whether the problem is accessory-specific or system-related.

If the Problem Persists

If all external causes have been ruled out, the issue is likely internal (sensor, actuator, or calibration system failure).

Remove the device from service, label it Out of Service, and send for repair or bench evaluation.

Clinical Use Tip

Do not troubleshoot compression or force-related issues with a patient in the unit. Always transfer the patient before testing paddle movement or calibration.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Compression paddle error displayed and system would not allow proper compression."

Cause

What was observed during troubleshooting.

Example:
"Compression paddle not fully seated, preventing proper sensor recognition."

Resolution

What action was taken.

Example:
"Reseated paddle and verified normal operation with successful compression test."

Helpful Details to Include (If Known)

  • Paddle type used
  • Whether alternate paddle was tested
  • Error message wording
  • Paddle movement behavior
  • Presence of debris or obstruction
  • Calibration attempt results
  • Final system status

Final Thought

Start with the paddle itself—most issues are related to installation, alignment, or obstruction. Maintain a logical approach and prioritize patient safety. Escalate when internal calibration or sensor issues are suspected, and document clearly to support efficient repair.

That is successful troubleshooting.
