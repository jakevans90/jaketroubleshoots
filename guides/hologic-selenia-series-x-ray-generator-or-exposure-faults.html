Hologic Selenia Series

X-ray generator or exposure faults

Asset Type

Mammography System

Manufacturer

Hologic

Model

Selenia Series

What This Guide Helps With

Identifies external causes of exposure failures such as interlocks, power issues, positioning, or readiness conditions preventing X-ray generation.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Confirm no patient is in compression and no exposure is active.
  • If a patient is present, safely release compression and move them to another unit.
  • Why: Prevents unintended radiation exposure and ensures safe troubleshooting conditions.

Verify System Ready Status

  • Check the system interface for “Ready” or equivalent status.
  • Look for warning messages or system faults.
  • Expected Outcome: System indicates ready for exposure.
  • Why: The generator will not fire unless all system conditions are satisfied.

Check Emergency Stop (E-Stop) Buttons

  • Inspect all E-stop buttons (room, gantry, console). Ensure they are fully released.
  • Expected Outcome: Releasing E-stop clears interlock faults and allows exposure.
  • Why: Engaged E-stops disable generator output.

Inspect Door and Room Interlocks

  • Verify the exam room door is fully closed and latched.
  • Check any interlock indicators on the system.
  • Expected Outcome: Closed door clears exposure inhibit condition.
  • Why: Open doors or interlock faults prevent X-ray generation for safety.

Verify Compression and Paddle Positioning

  • Ensure the compression paddle is properly attached and engaged.
  • Confirm adequate compression is applied if required by the system.
  • Expected Outcome: Proper positioning allows exposure enablement.
  • Why: Many systems require correct compression and positioning before enabling exposure.

Confirm Technique and Exam Setup

  • Review selected exam type, technique factors, and detector readiness.
  • Ensure no invalid settings or incomplete setup steps exist.
  • Expected Outcome: Valid technique settings allow exposure attempt.
  • Why: Incorrect or incomplete exam setup can block generator activation.

Check System Power Stability

  • Verify power to the system:
  • Main system power indicators
  • Generator cabinet (if visible)
  • No recent power interruptions
  • If available, confirm UPS status or check for recent power alarms.
  • Expected Outcome: Stable power with no interruptions.
  • Why: Generator faults often occur with unstable or insufficient power.

Inspect Cables and Connections (External Only)

  • Check visible cables between:
  • Generator cabinet
  • Detector
  • System console
  • Look for loose, damaged, or disconnected cables. Reseat if necessary.
  • Expected Outcome: Secure connections restore communication if fault was connection-related.

Perform a Controlled System Restart

  • Shut down the system using proper procedure.
  • Wait 2–3 minutes, then power back on.
  • Expected Outcome: Transient generator or interlock faults may clear.
  • Why: Resets system synchronization and clears temporary faults.

Attempt Test Exposure (No Patient)

  • Perform a system test exposure if allowed by protocol.
  • Expected Outcome: Successful exposure confirms resolution.
  • If resolved: Return system to service.

If the Problem Persists

If all external checks have been completed and the system still cannot generate exposure, the issue is likely internal (generator, HV components, or internal interlocks).

Remove the system from service

Label as Out of Service

Send for repair or advanced bench evaluation

Clinical Use Tip

Never troubleshoot exposure issues with a patient in the system. Always move the patient to another unit before attempting resets or test exposures. Maintain therapy continuity by using a backup device.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"System would not allow exposure; “generator fault” displayed during exam setup."

Cause

What was observed during troubleshooting.

Example:
"Room door interlock not fully engaged, preventing exposure enablement."

Resolution

What action was taken.

Example:
"Secured door latch, verified system ready status, and confirmed successful test exposure."

Helpful Details to Include (If Known)

  • E-stop status verified
  • Door interlock checked
  • Compression paddle engagement confirmed
  • Power stability verified
  • Cables inspected/reseated
  • Error messages observed
  • Test exposure result
  • Final system status (returned to service or removed)

Final Thought

Exposure faults should always be approached by verifying safety interlocks, system readiness, and power stability before suspecting internal failure. A structured approach protects patients, reduces downtime, and avoids unnecessary escalation. Proper documentation ensures clarity and supports future troubleshooting.

That is successful troubleshooting.
