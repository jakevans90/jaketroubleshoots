Baxter SIGMA Spectrum

Error: 303 Bad upper US rdg

Asset Type

Infusion Pump

Manufacturer

Baxter

Model

SIGMA Spectrum

What This Guide Helps With

This guide assists with troubleshooting Error 303 Bad upper US rdg, which indicates an invalid upper ultrasonic sensor reading. This condition can interrupt infusion, prevent pump startup, or cause inaccurate fluid and air-in-line detection.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Remove the pump from active patient use.
  • Transfer infusion to a functioning device.
  • Ensure therapy continues without interruption.

Verify the Reported Error and Behavior

  • Power on the pump and confirm Error 303 Bad upper US rdg is displayed.
  • Observe when the error occurs (startup, set loading, or infusion).
  • Check for symptoms such as failure to start infusion or repeated alarms.

Check IV Set Installation and Compatibility

  • Verify the correct Baxter-approved IV set is installed.
  • Ensure tubing is properly seated in the pump channel.
  • Pay close attention to alignment in the upper sensor area.
  • Reseat tubing fully.

Inspect Tubing Condition

  • Check for air bubbles in the tubing.
  • Inspect for kinks, compression, or deformation.
  • Look for moisture or residue on the tubing.
  • Replace the IV set if any issues are found.

Clean the Sensor Area

  • Open the pump door and inspect the upper ultrasonic sensor area.
  • Remove any fluid, residue, or debris.
  • Clean with approved wipes and allow to fully dry.

Check Door Closure and Latch

  • Ensure the pump door is fully closed and securely latched.
  • Confirm proper pressure is applied to the tubing.

Power Cycle the Pump

  • Turn the pump off.
  • Wait 10 seconds.
  • Restart the pump and recheck for the error.

Swap IV Set and Retest

  • Install a new, known-good IV set.
  • Test pump operation again.
  • Determine if the issue was accessory-related.

If the Problem Persists

If Error 303 continues after replacing the IV set, cleaning the sensor area, and verifying setup, external causes have been ruled out.

Remove the device from service, label it Out of Service, and send it for repair or bench evaluation. Persistent errors indicate likely internal ultrasonic sensor or ADC circuitry failure.

Clinical Use Tip

Never troubleshoot on an active patient. Always transition the patient to a backup infusion device before addressing ultrasonic sensor errors.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Pump displaying Error 303 Bad upper US rdg; infusion would not start."

Cause

What was observed during troubleshooting.

Example:
"Invalid ultrasonic sensor reading persisted after IV set replacement and cleaning, indicating internal sensor failure."

Resolution

What action was taken.

Example:
"Removed from service and sent for repair."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Ultrasonic sensor errors should always be taken seriously. Start with tubing setup and cleanliness, but recognize when persistent faults point to internal failure. Proper escalation ensures patient safety and device reliability.

That is successful troubleshooting.
