Drager Evita V500

Alarm: Tube type / I.D. incorrect mismatch between configured tube settings and actual airway size

Asset Type

Ventilator

Manufacturer

Drager

Model

Evita V500

What This Guide Helps With

This guide addresses the alarm indicating a mismatch between the configured tube type or inner diameter (I.D.) and the actual airway device. This condition can lead to inaccurate pressure and volume delivery, impacting ventilation performance and patient safety.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Assess the patient’s ventilation status immediately
  • If ventilation is unstable, transition to a backup ventilator or bag-valve mask
  • Do not continue troubleshooting on an unstable patient

Verify the Reported Alarm and Behavior

  • Confirm the alarm message: “Tube type / I.D. incorrect”
  • Observe when the alarm occurs (startup, setup, or during ventilation)
  • Compare delivered pressures and volumes to expected values
  • Check for additional related alarms

Check Tube Type and I.D. Settings

  • Access ventilator configuration settings
  • Verify selected tube type (endotracheal or tracheostomy)
  • Confirm the configured inner diameter matches the intended tube

Verify Actual Airway Device in Use

  • Inspect the airway device connected to the patient
  • Check labeling on the tube for inner diameter size
  • Confirm tube size with clinical staff if needed

Correct Configuration as Needed

  • Adjust ventilator settings to match the actual tube type and size
  • Ensure changes are confirmed and saved
  • Monitor for alarm clearance and stable ventilation values

Check for Configuration Input Issues

  • Re-enter tube settings if already correct
  • Confirm system accepts and applies changes
  • Verify no user input errors occurred during setup

Perform System Check

  • Run a system test or device check
  • Confirm the ventilator validates configuration and clears the alarm

If the Problem Persists

If correct tube settings and airway verification have been completed and the alarm remains:

  • External and configuration causes have been ruled out
  • Remove the device from service
  • Label the device as Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot configuration-related alarms on an active patient. Always move the patient to a backup ventilator before making adjustments to ensure safe and stable ventilation.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Tube type / I.D. incorrect alarm displayed during ventilation with inconsistent delivered volumes."

Cause

What was observed during troubleshooting.

Example:
"Configured tube I.D. did not match actual endotracheal tube size in use."

Resolution

What action was taken.

Example:
"Verified airway device size, updated ventilator settings to correct I.D., alarm cleared and device returned to service."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Configuration mismatches are a common but critical cause of ventilator alarms. Always verify the real-world setup before assuming device failure. Accurate configuration ensures safe and effective ventilation.

That is successful troubleshooting.
