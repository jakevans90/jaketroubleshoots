BD Alaris

Error: 241.4150 Patient-Side Pressure Sensor Voltage too high

Asset Type

Infusion Pump

Manufacturer

BD

Model

Alaris

What This Guide Helps With

This guide addresses Error 241.4150, indicating high voltage from the patient-side pressure sensor. This condition can lead to false downstream occlusion alarms, inaccurate pressure monitoring, or unexpected infusion interruption.

Step-by-Step Troubleshooting

1. Ensure Patient Safety First

  • Stop the infusion if active
  • Transfer therapy to a backup infusion pump
  • Remove the affected pump from patient use

2. Verify the Reported Error and Behavior

  • Power on the pump and confirm Error 241.4150 is displayed
  • Observe when the error occurs (startup vs infusion)
  • Check for repeated occlusion alarms or infusion stoppage

3. Check for Downstream Occlusion or Restriction

  • Inspect for closed clamps
  • Check for kinked or compressed tubing
  • Assess IV site for infiltration or blockage
  • Correct any restrictions and retest

4. Verify IV Set Type and Proper Loading

  • Confirm use of BD-approved Alaris administration set
  • Ensure tubing is properly seated in the pump channel
  • Check for twisting or pinching of tubing
  • Close and latch the pump door securely

5. Inspect Pump Channel and Pressure Interface

  • Open the pump door and inspect the channel
  • Look for fluid ingress, residue, or debris
  • Clean externally per facility policy if needed

6. Power Cycle the Pump

  • Turn the pump off
  • Wait 10–15 seconds
  • Power back on and observe for error recurrence

7. Replace IV Set and Retest

  • Install a new, known-good IV set
  • Run the pump and monitor for error clearance

8. Determine if Issue Persists

  • If the error continues after all external checks
  • Suspect internal pressure sensor or circuitry fault

If the Problem Persists

If all external causes have been ruled out and the error continues, remove the device from service, label it as Out of Service, and send it for repair or bench evaluation.

Clinical Use Tip

Never troubleshoot a pump with active pressure sensor errors on a patient. Always transition the patient to a backup device before beginning troubleshooting.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Pump displaying Error 241.4150 with repeated occlusion alarms and stopping infusion."

Cause

What was observed during troubleshooting.

Example:
"No downstream occlusion found, IV set properly loaded, channel clean. Error persisted, indicating likely pressure sensor fault."

Resolution

What action was taken.

Example:
"Removed pump from service, labeled out of service, and sent for repair."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Always prioritize external causes like occlusions and setup issues before suspecting internal faults. Persistent sensor voltage errors indicate deeper system issues that require escalation and proper documentation.
