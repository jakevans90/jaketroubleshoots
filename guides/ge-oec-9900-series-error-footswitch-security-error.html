GE Healthcare OEC 9900 Series

Error: FOOTSWITCH SECURITY ERROR

Asset Type

C-Arm

Manufacturer

GE Healthcare

Model

OEC 9900 Series

What This Guide Helps With

This guide helps troubleshoot FOOTSWITCH SECURITY ERROR on a GE Healthcare OEC 9900 Series C-Arm. This error means the X-ray security line does not match the X-ray control signal. It may involve the left or right X-ray control path and can cause the system to shut down or prevent X-ray operation.

The goal is to check external causes first, including the footswitch, hand control, cable condition, connector seating, accessory compatibility, and visible contamination before escalating the unit for service evaluation.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Confirm the C-Arm is not being used for an active procedure before troubleshooting.
  • If imaging is needed, have clinical staff switch to another approved C-Arm or imaging system.
  • Do not repeatedly attempt X-ray activation while the FOOTSWITCH SECURITY ERROR is active.
  • Remove the device from clinical use if it shuts down, will not enable X-ray, or reports the error during startup or exposure attempts.

Verify the Reported Error and Behavior

  • Power the system on and confirm the displayed message is FOOTSWITCH SECURITY ERROR.
  • Note whether the error appears during startup, when the footswitch is connected, or when X-ray is attempted.
  • Ask whether the issue happens with the left pedal, right pedal, hand switch, or all X-ray controls.
  • Document whether the system shuts down immediately or only after an exposure command.

Inspect the Footswitch for Damage or Sticking

  • Inspect the footswitch housing, pedal movement, cable strain relief, and connector.
  • Look for cracked housing, fluid intrusion, impact damage, crushed cable, bent pins, or loose connector hardware.
  • Verify that the left and right footswitch controls return normally and are not mechanically stuck.
  • Do not use the footswitch if it appears damaged, contaminated, or unreliable.

Check Footswitch Connection and Seating

  • Power the system down according to facility procedure.
  • Disconnect and reconnect the footswitch connector.
  • Inspect for bent pins, contamination, corrosion, poor seating, or damaged locking hardware.
  • Power the system back on and check whether the error returns.
  • If the error clears, perform an approved operational check before returning the unit to service.

Test With a Known-Good Footswitch if Available

  • Connect a known-good compatible OEC footswitch if available.
  • Power the system on and verify whether the FOOTSWITCH SECURITY ERROR returns.
  • Test X-ray enable behavior only under approved safe test conditions.
  • If the error clears with a known-good footswitch, the original footswitch or its cable is the likely cause.

Check the X-Ray Hand Control if Applicable

  • Inspect the X-ray hand switch or hand control if installed.
  • Look for damaged cable, loose connector, stuck exposure button, contamination, or physical stress.
  • Disconnect nonessential external controls one at a time if allowed by facility procedure and retest startup behavior.
  • Determine whether the fault follows one control accessory or remains present regardless of the connected control.

Inspect External Cable Routing and Strain Points

  • Trace the footswitch and X-ray control cables from the connector to the control device.
  • Check for pinching under wheels, crushed insulation, sharp bends, stretched cable sections, or damaged strain relief.
  • Gently move the cable while observing for intermittent error behavior, without activating X-ray.
  • Remove any damaged cable or accessory from clinical use.

Confirm Correct Accessory Configuration

  • Verify the connected footswitch or X-ray control is compatible with the GE Healthcare OEC 9900 Series.
  • Ask whether the issue began after an accessory swap, repair, loaner footswitch, cleaning event, or cable change.
  • Remove any questionable, mismatched, or unapproved accessory from use.
  • Confirm only approved X-ray controls are connected before retesting.

Check for Contamination or Fluid Intrusion

  • Inspect the footswitch, connector, and cable ends for moisture, dried residue, corrosion, or cleaning chemical buildup.
  • Do not use the footswitch if fluid intrusion is suspected.
  • Allow only minor external connector moisture to dry according to facility procedure.
  • Escalate accessories with suspected internal contamination for repair or replacement.

Isolate Device-Side Failure

  • If the error remains with known-good compatible controls, suspect a C-Arm-side issue.
  • Document that the footswitch, hand control, connectors, cable condition, accessory compatibility, and known-good control testing were checked.
  • Do not open panels or perform board-level troubleshooting during basic response unless trained and authorized.
  • Escalate for service-level troubleshooting of the control input path, connector interface, internal wiring, or X-ray control/security circuitry.

If the Problem Persists

If the FOOTSWITCH SECURITY ERROR remains after checking the footswitch, hand control, connector seating, cable routing, accessory compatibility, contamination, and known-good control devices, external causes have been ruled out as much as possible during first-response troubleshooting.

Remove the C-Arm from service, label it Out of Service, and send it for repair, bench evaluation, or vendor service. Persistent X-ray security-line mismatch may indicate an internal wiring issue, connector interface failure, control circuitry fault, or exposure-control safety hardware problem.

Clinical Use Tip

Never troubleshoot on an active patient. Move the patient or procedure to a backup imaging device first whenever possible.

A C-Arm with an unresolved FOOTSWITCH SECURITY ERROR should not be trusted for clinical imaging because exposure control may be unavailable, intermittent, or unsafe.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"GE Healthcare OEC 9900 Series C-Arm displays FOOTSWITCH SECURITY ERROR and system shuts down or will not enable X-ray when the footswitch or X-ray control is connected or activated."

Cause

What was observed during troubleshooting.

Example:
"Footswitch security line did not match the X-ray control signal. Footswitch, hand control, connector seating, cable condition, accessory compatibility, and visible contamination were checked. Fault remained with known-good compatible control, indicating likely control/security signal path issue."

Resolution

What action was taken.

Example:
"Removed C-Arm from clinical use, inspected external X-ray controls and connectors, tested with known-good compatible footswitch where available, labeled unit Out of Service, and escalated for service evaluation."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

FOOTSWITCH SECURITY ERROR should be treated as an X-ray control safety issue, not a nuisance message. Start with the external footswitch, hand control, cables, connectors, and accessory compatibility, then escalate when the security mismatch remains. Good troubleshooting protects the patient, the operator, and the repair process.

That is successful troubleshooting.
