ICU Medical Plum 360

Alarm: Keypad locked. Enter code to disable.

Asset Type

Infusion Pump

Manufacturer

ICU Medical

Model

Plum 360

What This Guide Helps With

This guide helps resolve the alarm condition “Keypad locked. Enter code to disable.” which prevents user interaction with the pump controls. This condition can restrict the ability to adjust or stop infusion therapy if not addressed promptly.

Step-by-Step Troubleshooting

1. Ensure Patient Safety First

  • If infusion cannot be adjusted or stopped, transition therapy to a backup infusion device
  • Confirm the patient continues receiving prescribed therapy without interruption

2. Verify the Reported Alarm and Behavior

  • Confirm the alarm message is displayed: “Keypad locked. Enter code to disable.”
  • Check if keypad inputs are unresponsive
  • Determine if infusion is still running or paused
  • Note when the alarm occurred (after STOP press, door open, or during operation)

3. Attempt Keypad Unlock

  • Enter the facility-approved keypad unlock code
  • Use authorized clinical or biomed override code if applicable
  • Confirm keypad functionality returns after entry

4. Check for Accidental Lock Activation

  • Determine if STOP was pressed while keypad lock was active
  • Check if the cassette door was opened during a locked state
  • Review device settings for keypad lock enablement

5. Verify Keypad Function After Unlock

  • Test multiple keypad inputs
  • Navigate menus and confirm responsiveness
  • Simulate start/stop functions if safe to do so

6. Power Cycle if Unlock Fails

  • Turn the pump OFF
  • Disconnect from AC power briefly
  • Restart the device and reassess keypad response

If the Problem Persists

If the keypad cannot be unlocked or remains unresponsive after these steps, external causes have been ruled out. Remove the device from service, label it Out of Service, and send for repair or bench evaluation. Persistent lock conditions may indicate keypad interface or control board failure.

Clinical Use Tip

Never troubleshoot or rely on a device with a locked or unresponsive keypad during active patient use. Always transfer the patient to a functioning device first.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Keypad locked alarm displayed and unable to adjust infusion settings."

Cause

What was observed during troubleshooting.

Example:
"Keypad lock active due to STOP press or cassette door interaction while locked."

Resolution

What action was taken.

Example:
"Entered unlock code, restored keypad functionality, and verified normal operation."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Keypad lock alarms are often normal operational safeguards but must be resolved quickly to maintain control of infusion therapy. Always verify full functionality before returning the device to service and escalate appropriately if access cannot be restored.
