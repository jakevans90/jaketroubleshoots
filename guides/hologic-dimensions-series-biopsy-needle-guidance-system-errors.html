Hologic Dimensions Series

Biopsy / needle guidance system errors

Asset Type

Mammography System

Manufacturer

Hologic

Model

Dimensions Series

What This Guide Helps With

Identifies external causes of biopsy or needle guidance errors including attachment issues, positioning, calibration state, or system readiness conditions.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Confirm no biopsy procedure is in progress and no needle is engaged.
  • If necessary, move the patient to another unit.

Expected Outcome:
Safe environment with no risk of injury.

Observe Error Message and System Status

  • Note the exact biopsy or needle guidance error message.
  • Verify the system is fully in biopsy mode and initialized.

Expected Outcome:
System provides a clear error or transitions properly into biopsy mode.

Why:
Helps determine if the issue is workflow/setup related vs hardware-related.

Verify Biopsy Attachment Installation

  • Inspect the biopsy device attachment:
  • Fully seated and locked into position
  • Correct accessory for the procedure
  • No looseness or misalignment
  • Remove and reseat if needed.

Expected Outcome:
System recognizes the biopsy attachment and clears related errors.

Why:
Improper installation is a common cause of guidance system faults.

Inspect Mechanical Alignment and Mounting Area

  • Check for:
  • Misalignment between biopsy unit and gantry
  • Obstructions or debris in mounting interface
  • Physical damage to brackets or connectors

Expected Outcome:
Clear mounting interface allows proper alignment and sensor recognition.

Why:
Alignment issues can prevent accurate positioning and system readiness.

Check Cable Connections (If Applicable)

  • Inspect any visible cables between the biopsy unit and system:
  • Secure connections
  • No bent pins or damage
  • Proper routing without strain
  • Reseat connections if accessible.

Expected Outcome:
Restored communication between biopsy system and main unit.

Why:
Loose or damaged cables can cause communication or initialization errors.

Verify System Calibration / Homing Status

  • Check if the system requires:
  • Homing or initialization sequence
  • Calibration confirmation before biopsy use
  • Perform system homing if available.

Expected Outcome:
System completes initialization and clears positioning-related errors.

Why:
Biopsy guidance relies on accurate positional reference points.

Confirm Gantry Position and System Readiness

  • Ensure:
  • Gantry is in correct biopsy position
  • All system interlocks are satisfied (door, compression, etc.)

Expected Outcome:
System enters “ready” state for biopsy procedures.

Why:
Incorrect positioning or unmet interlocks will prevent guidance activation.

Restart Biopsy Application / System (If Needed)

  • Exit biopsy mode and re-enter.
  • If issue persists, perform a controlled system reboot.

Expected Outcome:
Software-related faults are cleared and system initializes correctly.

Why:
Temporary software faults can disrupt biopsy system communication.

If the Problem Persists

If all external causes have been ruled out, the issue is likely internal (sensor failure, positioning encoder issue, or biopsy control hardware fault).

Remove the device from service, label it Out of Service, and send for repair or further bench evaluation.

Knowing when to stop is proper troubleshooting and prevents unnecessary risk.

Clinical Use Tip

Never troubleshoot biopsy guidance issues with a needle engaged or a patient under compression. Always transition the patient to a safe alternative first.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Biopsy system would not initialize and displayed guidance system error."

Cause

What was observed during troubleshooting.

Example:
"Biopsy attachment not fully seated, preventing system recognition."

Resolution

What action was taken.

Example:
"Reseated biopsy attachment and verified proper system initialization; unit returned to service."

Helpful Details to Include (If Known)

  • Biopsy attachment reseated
  • Gantry position verified
  • Calibration/homing performed
  • Cable connections checked
  • Error message observed
  • System reboot performed
  • Final device status

Final Thought

Biopsy system errors are often related to setup, alignment, or system readiness rather than true hardware failure. Start with simple external checks, prioritize safety, and escalate appropriately when needed. Proper documentation ensures clarity and supports future troubleshooting.

That is successful troubleshooting.
