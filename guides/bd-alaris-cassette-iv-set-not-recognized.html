BD Alaris Infusion Pump

Cassette / IV Set Not Recognized

Asset Type

Infusion Pump

Manufacturer

BD

Model

Alaris

What This Guide Helps With

This guide assists Clinical Engineering in troubleshooting situations where a BD Alaris infusion pump does not recognize the installed cassette or IV set. The pump may display messages such as “Check IV Set,” “Set Not Installed,” or may prevent the infusion from starting even though the set appears properly loaded.

Many of these issues are caused by simple external factors such as improper set installation, debris around the sensor area, worn disposable sets, or door alignment issues rather than internal pump failure.

Step-by-Step Troubleshooting

Confirm the Pump Is Not Connected to an Active Patient

Before troubleshooting, ensure the patient has been safely moved to another infusion pump if therapy is required. Troubleshooting should not be performed while the device is delivering medication to a patient.

Verify the Correct IV Set Type

Confirm that the IV set installed is approved for use with the BD Alaris pump and matches the programmed infusion mode.

  • Incorrect or incompatible sets may not be recognized by the pump.

Open and Re-seat the IV Set

Open the pump door and fully remove the cassette or IV set. Reinstall the set carefully according to the loading diagram on the pump.

  • The cassette is fully seated in the cassette holder
  • The tubing is correctly routed through the pumping mechanism
  • There are no twists or misalignment in the set

Improper seating is one of the most common causes of this issue.

Inspect the Cassette Area for Debris or Fluid

Visually inspect the cassette detection area inside the pump.

  • Dried fluid residue
  • Dust or debris
  • Bent plastic guides

Debris or contamination may prevent the pump sensors from detecting the cassette correctly.

Inspect the Pump Door and Latch

Close the pump door slowly and ensure it fully latches.

  • Door not fully closing
  • Latch misalignment
  • Obstructions preventing proper closure

The pump must detect a fully closed door before recognizing the IV set.

Try a New IV Set

Remove the current disposable set and install a new IV set.

  • Disposable cassettes can occasionally have manufacturing defects, worn plastic features, or damage during storage or handling

Replacing the set helps rule out disposable component failure.

Restart the Pump

Power the pump off, wait a few seconds, then power it back on.

Restarting the pump allows the system to reinitialize its cassette detection sensors.

Test the Channel Module on Another PC Unit (If Available)

If the issue persists, connect the channel module to another Alaris PC Unit to determine if the issue follows the module.

This helps determine whether the problem is related to:

  • The channel module
  • The PC unit interface

If the Problem Persists

If the cassette or IV set still cannot be recognized after verifying proper set installation, replacing the disposable set, and confirming the door and sensor areas are clear, the issue is likely internal.

  • Cassette detection sensor failure
  • Door sensor failure
  • Mechanical alignment issues
  • Internal communication faults

At this point the device should be:

  • Removed from service
  • Labeled Out of Service
  • Sent for repair or bench evaluation

Recognizing when external causes have been ruled out is an important part of effective troubleshooting.

Clinical Use Tip

Do not troubleshoot infusion pump cassette recognition issues while the pump is connected to a patient. If medication delivery is required, immediately move the patient to a functioning infusion pump before investigating the problem.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
Nursing staff reported that the BD Alaris pump would not recognize the installed IV set and displayed a “Check IV Set” message.

Cause

What was observed during troubleshooting.

Example:
Inspection found the cassette was not fully seated in the pump channel and the door was not fully latched.

Resolution

What action was taken.

Examples:
Reinstalled IV set correctly and verified proper door closure. Pump recognized the set and passed functional testing.

Helpful Details to Include (If Known)

  • Outlet tested
  • Pump powered normally
  • IV set replaced
  • Cassette properly seated
  • Door latch verified
  • Sensor area clean
  • Alarm or error message displayed
  • Channel tested on alternate PC unit
  • Final device status (returned to service or sent for repair)

Final Thought

Effective troubleshooting begins with the most common and easily corrected issues. Verifying correct IV set installation, door closure, and disposable set condition often resolves cassette recognition problems quickly. When these external causes are ruled out, escalating the device for repair helps maintain patient safety and ensures reliable infusion therapy.

That is successful troubleshooting.
