STERIS STERIS AMSCO 4085

Error: CHARGER DEF

Asset Type

Surgical Table

Manufacturer

STERIS

Model

STERIS AMSCO 4085

What This Guide Helps With

Error: CHARGER DEF indicating charger failure, no battery charging, or loss of backup power due to fuse or charger malfunction. This condition prevents the table from maintaining battery charge and can eliminate backup operation during procedures.

Step-by-Step Troubleshooting

1. Ensure Patient Safety First

  • Remove the table from patient use if battery function is required or charging is not working
  • Transfer the patient to a functioning table if needed
  • Ensure no procedures rely on this table during troubleshooting

2. Verify the Reported Error and Behavior

  • Power on the table and confirm “CHARGER DEF” is displayed
  • Observe when the error occurs (startup or during use)
  • Check if the table charges when plugged into mains
  • Look for battery indicators showing no charge activity

3. Check Mains Power Supply

  • Confirm power cord is securely plugged in
  • Test wall outlet using a known working device
  • Verify table power indicator (green light) is illuminated

4. Inspect Power Cord and Connections

  • Inspect cord for damage, cuts, or loose connections
  • Ensure connector is fully seated at the table
  • Try a known good power cord if available

5. Check Fuse F1 (Charger Fuse)

  • Power off and unplug the table
  • Access fuse holder if externally accessible
  • Inspect fuse for continuity or visible damage
  • Replace with correct rated fuse if blown

6. Recheck Charging Function

  • Restore power to the table
  • Observe battery charging indicators
  • Confirm error clears

7. Isolate Charger vs Battery Issue

  • Check if batteries hold any charge
  • Observe if table runs only on mains power
  • Look for absence of charging indicators

If the Problem Persists

If mains power, fuse F1, and external connections are verified and CHARGER DEF remains, internal charger failure is likely.

  • Remove the table from service
  • Label Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot on an active patient. Always move the patient to a backup table before addressing charging or power-related issues.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Table displaying CHARGER DEF error and not charging batteries when plugged into mains."

Cause

What was observed during troubleshooting.

Example:
"Blown fuse F1 or failed charger circuit after verifying power and connections."

Resolution

What action was taken.

Example:
"Verified AC power and connections. Replaced fuse F1. If unresolved, removed table from service and sent for repair."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Charger-related errors should always be approached by verifying external power sources and protective components first. Once those are ruled out, escalation is the safest and most efficient path.

That is successful troubleshooting.
