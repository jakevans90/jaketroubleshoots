ICU Medical Plum 360

Alarm: Cassette test failure! Check set.

Asset Type

Infusion Pump

Manufacturer

ICU Medical

Model

Plum 360

What This Guide Helps With

This guide addresses the “Cassette test failure! Check set.” alarm, which prevents infusion from starting due to a detected issue during the cassette system check. This may involve improper cassette installation, occlusion, or air in the line.

Step-by-Step Troubleshooting

Ensure Patient Safety First

  • Stop the infusion immediately if active
  • Transfer therapy to a backup infusion device
  • Confirm patient continues receiving ordered therapy

Verify the Reported Alarm and Behavior

  • Power on the pump and confirm the alarm message
  • Determine when the alarm occurs (startup or infusion start)
  • Check if infusion is prevented
  • Look for additional alarms such as occlusion or air-in-line

Check Cassette Installation (Most Common Cause)

  • Open the cassette door
  • Verify correct Plum 360 cassette is installed
  • Ensure cassette is fully seated and aligned
  • Close the door firmly until latched

Inspect for Kinks or Occlusions in the Line

  • Trace tubing from pump to patient
  • Remove any kinks or closed clamps
  • Ensure fluid path is unobstructed
  • Check patient site if applicable

Check for Air in the Line

  • Inspect tubing for visible air bubbles
  • Backprime per procedure if air is present
  • Ensure cassette and tubing are fully primed

Open and Re-Close Cassette Door

  • Open the cassette door fully
  • Re-seat the cassette properly
  • Close the door securely

Replace the Administration Set

  • Remove the current cassette/set
  • Install a new compatible administration set
  • Reattempt cassette test

Stop When Resolved

  • Restart infusion once alarm clears
  • Monitor briefly for normal operation
  • Return device to service if functioning properly

If the Problem Persists

If all external causes (setup, occlusion, air, cassette) have been ruled out and the alarm continues, this indicates a likely internal cassette sensor or mechanism failure.

  • Remove the pump from service
  • Label as Out of Service
  • Send for repair or bench evaluation

Clinical Use Tip

Never troubleshoot cassette or line issues while connected to a patient. Always transfer infusion therapy to a backup device first.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Pump displaying 'Cassette test failure! Check set.' and will not start infusion."

Cause

What was observed during troubleshooting.

Example:
"Cassette reseated and line inspected with no occlusion or air found. Fault persisted indicating possible cassette or internal detection issue."

Resolution

What action was taken.

Example:
"Replaced administration set and retested. If unresolved, removed pump from service and sent for repair. Verified alternate pump in use."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Cassette test failures are commonly caused by setup or disposable issues, but persistent alarms indicate internal faults. Always prioritize patient safety and escalate appropriately when external causes are ruled out.

That is successful troubleshooting.
