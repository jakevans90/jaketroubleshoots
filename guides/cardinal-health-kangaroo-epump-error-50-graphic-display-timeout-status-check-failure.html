Cardinal Health Kangaroo ePump

Error: 50 Graphic display timeout (status check failure)

Asset Type

Feeding Pump

Manufacturer

Cardinal Health

Model

Kangaroo ePump

What This Guide Helps With

This guide helps troubleshoot Error 50 Graphic display timeout (status check failure), which can cause the screen to freeze, become unresponsive, or prevent safe operation of the feeding pump.

Step-by-Step Troubleshooting

1. Ensure Patient Safety First

  • Stop using the pump if the display is unresponsive or unreadable
  • Transfer feeding to a backup pump
  • Ensure patient nutrition therapy continues without interruption

2. Verify the Reported Error and Behavior

  • Power on the pump and observe the display
  • Confirm Error 50 appears
  • Note if the screen is frozen, blank, or partially responsive
  • Determine if the issue occurs at startup or during operation

3. Perform a Power Cycle

  • Turn the pump off
  • Disconnect from AC power
  • Remove the battery if accessible
  • Wait 30–60 seconds
  • Reconnect power and restart

4. Check Power Source and Stability

  • Plug the pump into a known-good outlet
  • Verify AC power indicator is functioning
  • Test operation on battery vs AC power

5. Inspect Display and Front Panel Condition

  • Check for cracked screen or visible damage
  • Look for signs of liquid intrusion
  • Press buttons to check for any response

6. Attempt Basic Function Input

  • Try pressing buttons to navigate menus
  • Attempt start/stop functions
  • Listen for audible feedback or motor response

7. Verify Function if Resolved

  • Confirm full menu navigation
  • Verify alarms and feeding functions operate normally
  • Return device to service only if fully functional

If the Problem Persists

If Error 50 continues after completing all external troubleshooting:

  • External causes have been ruled out
  • This indicates an internal display communication or control board failure
  • Remove the device from service
  • Label the device as Out of Service
  • Send the unit for repair or bench evaluation

Clinical Use Tip

Never troubleshoot or use a feeding pump on an active patient when the display is not fully functional. Always transfer the patient to a properly working device first.

Work Order Documentation (CCR Method)

CCR = Complaint, Cause, Resolution

Complaint

What was reported by the clinical staff.

Example:
"Pump displaying Error 50 with frozen or unresponsive screen during use."

Cause

What was observed during troubleshooting.

Example:
"Display timeout persists after power cycle; no response from interface indicating internal display or control board failure."

Resolution

What action was taken.

Example:
"Removed unit from service, labeled out of service, and sent for repair evaluation."

Helpful Details to Include (If Known)

  • Alarm behavior
  • Accessories swapped
  • Power behavior
  • Environmental factors
  • Indicator lights
  • Final device status

Final Thought

Error 50 indicates a failure in communication between the display and internal electronics. Always rule out simple power-related issues first, but escalate quickly when the user interface is compromised, as safe operation depends on full visibility and control.
